# 📱 モバイルコンテンツ自動ストックワークフロー

**目的**: コストゼロで、XやYouTubeなどのコンテンツを自動的にObsidianにストックし、後で学習できるようにする

**最終更新**: 2025-11-17

---

## 🎯 目標

1. ✅ **コストゼロ**（完全無料）
2. ✅ **自動ストック**（手動操作を最小化）
3. ✅ **学習管理**（後で忘れずに学習できる）

---

## 📋 全体アーキテクチャ

```
X / YouTube
    ↓
iOSショートカット（自動化）
    ↓
iCloud Drive（無料同期）
    ↓
Obsidian（デスクトップ + モバイル）
    ↓
00_Memo → 01_Inbox → 04_Memory
```

---

## 🔧 ステップ1: iCloud Driveで同期設定（無料）

### デスクトップでの設定

1. **VaultをiCloud Driveに移動**
   ```bash
   # 現在の場所を確認
   pwd
   # /Users/user/Desktop/obsidian-ttygtd
   
   # iCloud Driveに移動（またはコピー）
   # Finderで以下を開く:
   # ~/Library/Mobile Documents/com~apple~CloudDocs/
   ```

2. **ObsidianでVaultを再オープン**
   - 「Open another vault」→「Open folder as vault」
   - iCloud Drive内の`obsidian-ttygtd`を選択

### iPhoneでの設定

1. **Obsidianアプリをインストール**（App Store、無料）
2. アプリを起動
3. 「Open folder as vault」→「iCloud Drive」
4. `obsidian-ttygtd`フォルダを選択

✅ **これで同期完了！**（無料）

## 🤖 ステップ2: iOSショートカットで自動ストック

### 方法A: X（Twitter）の投稿をストック

#### ショートカット1: 「X投稿をストック」

**作成手順**:
1. iPhoneの「ショートカット」アプリを開く
2. 「+」で新規作成
3. 以下のアクションを追加：

```
1. テキストを取得
   - 入力: 「共有」から
   - 変数名: 「投稿URL」

2. テキストを取得
   - 入力: 「共有」から
   - 変数名: 「投稿テキスト」

3. 現在の日時を取得
   - フォーマット: YYYY-MM-DD HH:mm

4. テキストを結合
   - テキスト1: "---\n"
   - テキスト2: "title: X投稿ストック\n"
   - テキスト3: "tags: [x-twitter, stock, learning]\n"
   - テキスト4: "created: " + 日時 + "\n"
   - テキスト5: "url: " + 投稿URL + "\n"
   - テキスト6: "---\n\n"
   - テキスト7: "# X投稿ストック\n\n"
   - テキスト8: "**URL**: " + 投稿URL + "\n\n"
   - テキスト9: "**投稿内容**:\n" + 投稿テキスト + "\n\n"
   - テキスト10: "**ストック日時**: " + 日時 + "\n\n"
   - テキスト11: "## 学習メモ\n\n"
   - テキスト12: "- [ ] 読む\n"
   - テキスト13: "- [ ] 要点をまとめる\n"
   - テキスト14: "- [ ] 04_Memoryに移動\n"

5. ファイルを保存
   - ファイル名: "x-stock-YYYY-MM-DD-HHmm.md"
   - 保存先: iCloud Drive/obsidian-ttygtd/00_Memo/
```

**使い方**:
1. Xアプリでストックしたい投稿を開く
2. 「共有」ボタンをタップ
3. 「ショートカット」→「X投稿をストック」を選択
4. 自動的に`00_Memo/`に保存される

### 方法B: YouTube動画をストック

**作成手順**:
1. 「ショートカット」アプリで新規作成
2. 以下のアクションを追加：

```

   - 入力: 「共有」から
   - 変数名: 「動画URL」

2. 動画タイトルを取得（URLから抽出）
   - または手動入力プロンプト

3. 現在の日時を取得
   - フォーマット: YYYY-MM-DD HH:mm

4. テキストを結合
   - テキスト1: "---\n"
   - テキスト2: "title: YouTube動画ストック\n"
   - テキスト3: "tags: [youtube, stock, learning]\n"
   - テキスト4: "created: " + 日時 + "\n"
   - テキスト5: "url: " + 動画URL + "\n"
   - テキスト6: "---\n\n"
   - テキスト7: "# YouTube動画ストック\n\n"
   - テキスト8: "**タイトル**: " + 動画タイトル + "\n\n"
   - テキスト9: "**URL**: " + 動画URL + "\n\n"
   - テキスト10: "**ストック日時**: " + 日時 + "\n\n"
   - テキスト11: "## 学習メモ\n\n"
   - テキスト12: "- [ ] 視聴する\n"
   - テキスト13: "- [ ] 要点をまとめる\n"
   - テキスト14: "- [ ] 04_Memoryに移動\n"

5. ファイルを保存
   - ファイル名: "youtube-stock-YYYY-MM-DD-HHmm.md"
   - 保存先: iCloud Drive/obsidian-ttygtd/00_Memo/
```

**使い方**:
1. YouTubeアプリでストックしたい動画を開く
2. 「共有」ボタンをタップ
3. 「ショートカット」→「YouTube動画をストック」を選択
4. 自動的に`00_Memo/`に保存される

### 方法C: 汎用リンクストック（X、YouTube、その他すべて対応）

#### ショートカット3: 「リンクをストック」（推奨）

**作成手順**:
1. 「ショートカット」アプリで新規作成
2. 名前: 「リンクをストック」
3. 「共有シートで表示」をON
4. 以下のアクションを追加：

```
1. URLを取得
   - 入力: 「共有」から
   - 変数名: 「URL」

2. テキストを取得
   - 入力: 「共有」から（オプション）
   - 変数名: 「メモ」

3. 現在の日時を取得
   - フォーマット: YYYY-MM-DD HH:mm

4. URLからドメインを抽出
   - 例: twitter.com → "x-twitter"
   - 例: youtube.com → "youtube"
   - 例: その他 → "web"

5. テキストを結合
   - テキスト1: "---\n"
   - テキスト2: "title: リンクストック\n"
   - テキスト3: "tags: [" + ドメイン + ", stock, learning]\n"
   - テキスト4: "created: " + 日時 + "\n"
   - テキスト5: "url: " + URL + "\n"
   - テキスト6: "---\n\n"
   - テキスト7: "# リンクストック\n\n"
   - テキスト8: "**URL**: " + URL + "\n\n"
   - テキスト9: "**ストック日時**: " + 日時 + "\n\n"
   - テキスト10: "**メモ**: " + メモ + "\n\n"
   - テキスト11: "## 学習メモ\n\n"
   - テキスト12: "- [ ] 読む/視聴する\n"
   - テキスト13: "- [ ] 要点をまとめる\n"
   - テキスト14: "- [ ] 04_Memoryに移動\n"

6. ファイルを保存
   - ファイル名: "link-stock-YYYY-MM-DD-HHmm.md"
   - 保存先: iCloud Drive/obsidian-ttygtd/00_Memo/
```

**使い方**:
- **X**: 投稿を共有→「リンクをストック」
- **YouTube**: 動画を共有→「リンクをストック」
- **その他**: 任意のリンクを共有→「リンクをストック」

## 📚 ステップ3: 学習管理ワークフロー

### 週次レビューで整理

毎週日曜に、`00_Memo/`内のストックコンテンツを整理：

1. **01_Inboxに移動**
   - Cursorで整理（タグ追加、リンク追加）

2. **振り分け**
   - **学習済み・重要**: `04_Memory/`に移動
   - **今週学習予定**: `03_Input/`に移動
   - **不要**: `99_Archive/`に移動

### Dailyノートで学習記録

学習したら、Dailyノートに記録：

```markdown
## 📚 Learned Today
- [[X投稿ストック-2025-11-17]]を読んだ
  - 要点: ...
  - 関連: [[AI Tools]]
```

## 🎯 より高度な自動化（オプション）

### IFTTT / Zapier（無料プランあり）

**IFTTT無料プラン**:
- 月5アプレットまで（十分）
- Xの投稿を自動でObsidianに保存可能

**設定例**:
1. IFTTTアカウント作成（無料）
2. アプレット作成:
   - **If**: Xで「いいね」した投稿
   - **Then**: iCloud Driveにファイル作成
3. ファイル形式: Markdown
4. 保存先: `obsidian-ttygtd/00_Memo/`

⚠️ **注意**: IFTTTは無料プランでも使えますが、iOSショートカットの方が簡単で確実です。

## 🔧 トラブルシューティング

### 問題1: ショートカットが「共有」に表示されない

**解決策**:
- ショートカットの設定で「共有シートで表示」をONにする
- iPhoneを再起動

### 問題2: iCloud Driveに保存されない

**解決策**:
- iCloud Driveの同期を確認（設定→Apple ID→iCloud）
- 「ファイルを保存」アクションの保存先を確認
- パス: `iCloud Drive/obsidian-ttygtd/00_Memo/`

### 問題3: 同期が遅い

**解決策**:
- 手動同期: Obsidianアプリで「Pull changes」を実行
- Wi-Fi接続を確認
- しばらく待つ（iCloud Driveは数分かかる場合あり）
