# 契約会社・クライアント管理ガイド

## 📋 概要

複数の契約会社と契約し、各契約会社ごとに複数のクライアントがいる場合の議事録管理方法を説明します。

---

## 🗂️ 構造

```
05_Output/
├─ Projects/                    # プロジェクト単位（期限あり）
│  └─ @Active/
│     └─ [契約会社名]-[クライアント名]-[プロジェクト名]/
│        └─ 04-delivery/
│           └─ [日付]-[議題].md
│
└─ Areas/                       # 継続的エリア（期限なし）
   └─ Business/
      └─ Contract-Companies/    # 契約会社ごとの管理
         ├─ [契約会社名]/
         │  ├─ 00-契約会社概要.md
         │  ├─ 01-全議事録一覧.md
         │  └─ Clients/         # クライアントごとの管理
         │     └─ [クライアント名]/
         │        ├─ 00-クライアント概要.md
         │        └─ 01-議事録一覧.md
         └─ README.md
```

---

## 📝 管理ルール

### 1. プロジェクト命名規則

**形式**: `[契約会社名]-[クライアント名]-[プロジェクト名]`

**例**:
- `NTTDX-中企庁-補助金申請システム`
- `NTTDX-NICE-長野県デジタル化`
- `北海道チーム-北海道科学大学-地域課題解決プログラム`

### 2. 議事録の保存場所

**プロジェクト内**: `05_Output/Projects/@Active/[プロジェクト名]/04-delivery/`

**ファイル命名**: `[日付]-[議題].md`

**例**: `2025-11-17-社外ミーティングメモ.md`

### 3. 議事録のリンク管理

議事録を作成したら、以下の3箇所にリンクを追加：

1. **プロジェクト内**: `04-delivery/` フォルダに実際の議事録ファイルを保存
2. **クライアントの議事録一覧**: `Contract-Companies/[契約会社名]/Clients/[クライアント名]/01-議事録一覧.md` にリンクを追加
3. **契約会社の全議事録一覧**: `Contract-Companies/[契約会社名]/01-全議事録一覧.md` にリンクを追加

## 🔄 ワークフロー

### 新しい議事録を作成する場合

1. **プロジェクトの `04-delivery/` に議事録を作成**
   - テンプレート: `06_Templates/Meeting/議事録テンプレート-契約会社クライアント版.md`
   - ファイル名: `[日付]-[議題].md`

2. **クライアントの `01-議事録一覧.md` にリンクを追加**
   ```
   ## 2025年11月
   - [[05_Output/Projects/@Active/[プロジェクト名]/04-delivery/[議事録ファイル名]|議題]] - 2025-11-17
   ```

3. **契約会社の `01-全議事録一覧.md` にリンクを追加**
   ```
   ### [クライアント名]関連
   - [[05_Output/Projects/@Active/[プロジェクト名]/04-delivery/[議事録ファイル名]|議題]] - 2025-11-17
   ```

### 新しいクライアントを追加する場合

1. **フォルダを作成**
   ```
   Contract-Companies/[契約会社名]/Clients/[クライアント名]/
   ```

2. **必須ファイルを作成**
   - `00-クライアント概要.md` - クライアントの基本情報
   - `01-議事録一覧.md` - 議事録へのリンク集

3. **契約会社の `00-契約会社概要.md` の「クライアント一覧」に追加**

### 新しい契約会社を追加する場合

1. **フォルダを作成**
   ```
   Contract-Companies/[契約会社名]/
   ```

2. **必須ファイルを作成**
   - `00-契約会社概要.md` - 契約会社の基本情報
   - `01-全議事録一覧.md` - 全クライアントの議事録へのリンク集

3. **`Clients/` フォルダを作成**

## 💡 使い方のコツ

### 議事録を探す場合

1. **契約会社から探す**: `Contract-Companies/[契約会社名]/01-全議事録一覧.md`
2. **クライアントから探す**: `Contract-Companies/[契約会社名]/Clients/[クライアント名]/01-議事録一覧.md`
3. **プロジェクトから探す**: `Projects/@Active/[プロジェクト名]/04-delivery/`

### 議事録の整理

- **週次レビュー**: 各契約会社の `01-全議事録一覧.md` を確認
- **月次レビュー**: クライアントごとの議事録を整理
- **プロジェクト完了時**: プロジェクトを `@Completed` に移動
