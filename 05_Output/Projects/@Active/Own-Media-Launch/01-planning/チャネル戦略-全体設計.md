# チャネル戦略 - 全体設計

## 概要

DX/AIコンサルタントとしての専門性を発信し、案件獲得とコミュニティ形成を実現するための6チャネル統合戦略。各チャネルの役割を明確にし、コンテンツ循環モデルで効率的に運用する。

---

## 6チャネル役割分担表

| チャネル | 役割 | ターゲット | トーン | 投稿頻度 | 主要KPI |
|----------|------|------------|--------|----------|---------|
| **ブログ** | SEO集客・専門性の深掘り | 検索流入（研修担当・DX推進者） | 専門的・体系的 | 週1本 | 月間PV、検索順位 |
| **YouTube** | 信頼構築・実演解説 | AI/DXツール活用層 | 実践的・親しみやすい | 週1本 + ショート | 登録者数、視聴時間 |
| **X (Twitter)** | 認知拡大・日常接触 | エンジニア・クリエイター | カジュアル・鋭い洞察 | 毎日1-3投稿 | インプレッション、フォロワー |
| **LinkedIn** | BtoB案件獲得・ビジネス信頼 | 企業DX担当・経営層 | プロフェッショナル | 週2-3本 | 接続数、問い合わせ |
| **note** | 思想リーダーシップ・深い考察 | 思考を深めたい層 | 親密・知的 | 週1本 | スキ数、フォロワー |
| **メルマガ** | リード育成・関係深化 | 既存フォロワー・見込み客 | パーソナル・実用的 | 週1通 | 登録者数、開封率 |

## 各チャネル詳細戦略

### 1. ブログ（自社サイト）

**ポジション**: コンテンツのハブ。すべての深掘りコンテンツの最終的な蓄積先。

**コンテンツ方針**:
- SEOを意識した長文記事（3,000-8,000字）
- DX事例、AI活用ガイド、技術チュートリアル
- CTA（メルマガ登録、問い合わせ）を記事末尾に設置

**投稿頻度**: 週1本（月4本）

**運用ルール**:
- キーワードリサーチ → アウトライン → 執筆 → SEOチェック → 公開
- 公開後3日以内に他チャネルへ展開
- 月次でSearch Console分析、リライト対象を特定

**既存戦略との整合**: [[05_Output/Areas/Content-Creation/Blog-Writing/00-content-strategy|ブログコンテンツ戦略]] のカテゴリ（プロダクティビティ、AIツール比較、ケーススタディ）を継承・拡張

---

### 2. YouTube

**ポジション**: 信頼構築エンジン。顔が見える実演で専門性を証明。

**コンテンツ方針**:
- Cursor/AI開発ツールのTips（ショート）
- DX事例解説・プロジェクトVlog（ロング）
- ミニ講座シリーズ（体系的学習コンテンツ）

**投稿頻度**: 週1本（ロング）+ 週2-3本（ショート）

**運用ルール**:
- 台本→撮影→編集→公開 リードタイム7日
- ショートはブログやnote記事の核心部分を1分に凝縮
- 概要欄にブログ記事リンク・メルマガ登録リンクを必ず記載

**既存戦略との整合**: [[05_Output/Areas/Content-Creation/YouTube-Channel/00-channel-strategy|YouTubeチャンネル戦略]] のコンテンツ柱（Cursor Tips、プロジェクトVlog、ミニ講座）を継承

---

### 3. X (Twitter)

**ポジション**: 認知拡大・日常接触ポイント。短文で鋭い洞察を発信。

**コンテンツ方針**:
- AI/DXニュースへのコメント・解説
- ブログ・YouTube公開の告知スレッド
- 日常の気づき・学び（等身大の発信）
- コミュニティメンバーの紹介・リツイート

**投稿頻度**: 毎日1-3投稿（うち週1本はスレッド）

**運用ルール**:
- 朝（8:00-9:00）と夜（20:00-21:00）の投稿を基本
- スレッドはブログ記事の要約版として構成
- 週1本はコミュニティ紹介

**既存戦略との整合**: [[05_Output/Areas/Content-Creation/Social-Media/X-Twitter/content-calendar|Xコンテンツカレンダー]] の投稿ルール（週3本、コミュニティ紹介）を継承・強化

---

### 4. LinkedIn

**ポジション**: BtoB案件獲得の入り口。企業向けの信頼性構築。

**コンテンツ方針**:
- DXプロジェクト事例（匿名化）の共有
- AI活用の経営インパクト分析
- 研修・コンサルの知見共有
- 業界トレンドへのビジネス視点のコメント

**投稿頻度**: 週2-3本

**運用ルール**:
- ブログ記事をビジネス向けに再編集して投稿
- プロフィールに実績・サービス内容を明記
- コメント・つながり申請に48時間以内に対応

---

### 5. note

**ポジション**: 思想リーダーシップ。深い考察で「この人の考え方が好き」を作る。

**コンテンツ方針**:
- AI・テクノロジーの構造分析
- 教育・学習方法論
- 起業・独立のリアルな記録
- 表面的でない深掘り考察

**投稿頻度**: 週1本

**運用ルール**:
- [[05_Output/Areas/Content-Creation/Note-Writing/00-note戦略|note戦略]] の文章スタイルガイドに準拠
- シリーズ企画を四半期に1回実施
- 公開後はX、LinkedInで告知

**既存戦略との整合**: note戦略の3本柱（AI・テクノロジー考察、教育・学習方法論、働き方・生き方）を4ピラー体制に拡張

---

### 6. メルマガ

**ポジション**: リード育成・関係深化。最もパーソナルな接点。

**コンテンツ方針**:
- 週次ダイジェスト（今週のブログ・YouTube・noteまとめ）
- メルマガ限定の裏話・深掘りコラム
- セミナー・サービスの先行案内
- 読者限定のテンプレート・チェックリスト配布

**投稿頻度**: 週1通（金曜配信）

**運用ルール**:
- 件名はベネフィット訴求（「○○する方法」「○○がわかる」）
- 本文は1,000-2,000字、読了5分以内
- CTAは1つに絞る（記事誘導 or サービス案内）
- 月次で開封率・クリック率を分析

## コンテンツ循環モデル

1つの素材を6チャネルに効率的に展開する循環フロー。

### 正方向の循環（メインフロー）

```
ブログ記事（長文・SEO最適化）
  │
  ├──→ YouTube動画（実演解説・図解）
  │      └──→ YouTubeショート（核心1分）
  │
  ├──→ Xスレッド（要約5-10ツイート）
  │      └──→ X単発投稿（核心フレーズ）
  │
  ├──→ LinkedIn投稿（ビジネス向けリライト）
  │
  ├──→ note記事（考察・深掘り版）
  │
  └──→ メルマガ（ダイジェスト + 限定コラム）
```

### 逆方向の循環（フィードバックループ）

```
Xでの反応が高い投稿
  └──→ ブログ記事に展開（深掘り）

YouTube視聴者コメント
  └──→ Q&A記事・FAQ更新

メルマガ読者の質問
  └──→ コンテンツテーマに反映

note記事への反響
  └──→ シリーズ化・YouTube動画化
```

### 週次コンテンツ制作フロー

| 曜日 | アクション |
|------|-----------|
| **月** | ブログ記事の企画・アウトライン作成 |
| **火** | ブログ記事の執筆・完成 |
| **水** | YouTube台本作成・撮影 |
| **木** | 動画編集、note記事執筆 |
| **金** | メルマガ作成・配信、LinkedIn投稿 |
| **土** | X展開（スレッド作成）、来週の企画 |
| **日** | 休息 or インプット（書籍、記事、動画） |
