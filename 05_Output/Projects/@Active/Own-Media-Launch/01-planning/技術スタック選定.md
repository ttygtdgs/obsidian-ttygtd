# 技術スタック選定

## 概要

オウンドメディアの技術基盤を選定する。ブログ/Webサイトを中心に、動画編集、SNS管理、メール配信、分析ツールの比較検討を行い、推奨構成を提案する。

### 各プラットフォームの詳細

#### Next.js + Vercel（推奨）

**メリット**:
- [[04_Memory/Technical/Programming/Next.js/Next.jsとは何か|Vault内にNext.js知識が体系化されている]]（16ノート）
- App Router + Server Componentsで最新のWeb体験を提供
- [[04_Memory/Technical/Programming/Next.js/Next.jsのビルドとデプロイ|Vercelへのデプロイが極めて簡単]]
- MDX対応でObsidianのMarkdownを直接活用可能
- ISR（Incremental Static Regeneration）でSEOとパフォーマンスを両立
- Vercel Analyticsで基本的な分析も可能

**デメリット**:
- 初期構築に開発時間が必要（2-3週間）
- CMS機能は自前実装かヘッドレスCMS連携が必要
- Reactの知識が前提

**推奨構成**:
```
フレームワーク: Next.js 15 (App Router)
ホスティング:   Vercel (Hobby or Pro)
CMS:           Obsidian → GitHub → Next.js (MDXベース)
スタイリング:   Tailwind CSS
コンテンツ:     MDX (Markdown + JSXコンポーネント)
OGP画像:       @vercel/og (動的生成)
検索:          Algolia or Pagefind (静的検索)
```

**月額コスト**: $0-20（Vercel Hobby: 無料、Pro: $20/月）

#### WordPress

**メリット**:
- 世界最大のCMS、プラグインエコシステムが充実
- Yoast SEOなどSEO対策が容易
- 非エンジニアでも運用可能

**デメリット**:
- セキュリティリスク（定期的な更新が必須）
- プラグインの肥大化によるパフォーマンス劣化
- サーバー管理が必要

#### Astro + Cloudflare Pages

**メリット**:
- コンテンツサイトに最適化された設計
- ビルド速度が非常に速い
- Cloudflare Pagesの無料枠が太い
- Island Architectureで部分的にインタラクティブ

**デメリット**:
- Next.jsほどエコシステムが成熟していない
- Vault内に知見がない（学習コスト）

#### note pro

**メリット**:
- 即座に始められる、構築不要
- noteプラットフォームの回遊からの流入
- 独自ドメイン対応（pro版）

**デメリット**:
- カスタマイズ性が極めて低い
- プラットフォーム依存（データポータビリティ）
- 月額コストが高い（¥8,000/月）
- SEO最適化の自由度が低い

## 推奨構成: Next.js + Vercel

既存のNext.js知識資産と技術的自由度を考慮し、**Next.js + Vercel** を推奨する。

### アーキテクチャ

```
[Obsidian Vault]
    │ (Markdownファイル)
    ▼
[GitHub Repository]
    │ (git push → Webhook)
    ▼
[Next.js Build] (Vercel)
    │ (MDX変換 + SSG/ISR)
    ▼
[Vercel Edge Network]
    │ (CDN配信)
    ▼
[ユーザーのブラウザ]
```

### 実装ステップ

1. `create-next-app` でプロジェクト作成（TypeScript + Tailwind）
2. MDXベースのブログシステム構築（contentlayer or next-mdx-remote）
3. SEO設定（メタデータ、sitemap.xml、robots.txt）
4. OGP画像自動生成（@vercel/og）
5. RSS フィード生成
6. メルマガ登録フォーム設置
7. GA4 / Search Console連携
8. 独自ドメイン設定

| サービス | 無料枠 | 有料プラン | 特徴 |
|----------|--------|-----------|------|
| **ConvertKit** | 1,000登録者 | $29/月~ | クリエイター向け、自動化が強い |
| **Mailchimp** | 500登録者 | $13/月~ | 多機能、テンプレート豊富 |
| **Resend** | 3,000通/月 | $20/月~ | 開発者向け、API first |

- タグ・セグメント管理が優秀
- 自動化（シーケンス）でリード育成フロー構築可能
- 1,000登録者までは無料

| ツール | コスト | 特徴 | 推奨度 |

## SNS管理ツール

| ツール | コスト | 対応SNS | 特徴 |
|--------|--------|---------|------|
| **Buffer** | 無料-$6/月/ch | X, LinkedIn, Instagram等 | シンプルなスケジュール投稿 |
| **Typefully** | $12/月 | X, LinkedIn, Threads | Xスレッド作成に特化、分析機能 |
| **Hootsuite** | $99/月~ | 全主要SNS | 大規模向け、高額 |
| **手動運用** | 無料 | 全チャネル | 初期はこれで十分 |

**推奨**: 初期は**手動運用** → フォロワー増加に伴い **Buffer** or **Typefully** を導入
- 初期（~500フォロワー）: 手動投稿でリアクションを体感
- 成長期（500~）: Bufferでスケジュール投稿を効率化
- Xスレッド多用時: Typefullyでスレッド作成を効率化
