### スライド6: YouTubeの答え（3:30-4:15）

```
【話すこと】
「YouTubeの仮説はこうです。

昔は『クリック数』を重視していました。
でも、釣りタイトルやサムネ詐欺が増えて、満足度が下がった。

そこで気づいたんです。
『クリック数』じゃなく『視聴時間』こそが満足度の指標だ、と。

だから取得するデータはこうなります。

・視聴時間：どこまで見たかで満足度を測る
・スキップ位置：つまらない箇所を特定する
・繰り返し視聴：価値の高いコンテンツを発見する
・いいね/低評価：明示的なフィードバック

『何を測るか』を変えたら、結果も変わったんです。」
```

## 補足: トラブルシューティング対応

### よくある質問と回答

**Q: Googleアカウントでログインできない**
```
A: シークレットモード（プライベートブラウジング）で試してみてください。
   それでもダメなら、別のブラウザを試すか、
   スマホのGmailアプリでログインして、PCと同期されているか確認してください。
```

**Q: CSVファイルが読み込めない**
```
A: ファイルの文字コードがUTF-8になっているか確認してください。
   Excelで保存するときに「CSV UTF-8」を選ぶ必要があります。
   また、列名に「/」や「\」などの特殊文字があると
   エラーになることがあります。
```

**Q: グラフが表示されない**
```
A: ディメンションと指標の設定を確認してください。
   ディメンションには「何を軸にするか」（例: 選手名）
   指標には「何の数値を見るか」（例: 得点）
   を設定する必要があります。
```

**Q: 日本語が文字化けする**
```
A: データソースの設定を開いて、
   文字コードの設定を確認してください。
   「UTF-8」になっていない場合は変更してください。
```
