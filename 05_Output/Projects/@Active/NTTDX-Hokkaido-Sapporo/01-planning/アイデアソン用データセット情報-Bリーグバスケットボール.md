# アイデアソン用データセット情報 - Bリーグバスケットボールデータ

## 📋 概要

**データ種別**: Bリーグ試合データ・イベントデータ・成績データ・選手プロフィールデータ  
**提供元**: さっぽろ市データ圏  
**データ期間**: 未確認（要確認）  
**データ形式**: CSV/JSON（推測、要確認）  
**データサイズ**: 未確認（要確認）

## 🎯 活用可能性

このデータセットは以下のような活用が考えられます：
- **選手の動きを可視化**: シュート位置、プレー位置をコート上で可視化
- **プレーの分析**: シュート、アシスト、リバウンドなどのプレーを数値で分析
- **マーケティング施策**: 観客動員予測、人気カード分析、イベント企画
- **コンテンツ生成**: インパクトのある映像コンテンツ、SNS投稿自動生成
- **スポーツバー・飲食店舗との連携**: リアルタイムデータ可視化、選手紹介コンテンツ

#### 1.2 アクション情報（3項目）

| 項目名 | データ型 | 説明 | 活用可能性 |
|--------|----------|------|------------|
| アクション1 | integer | 該当プレイのアクションID（主要アクション） | プレー種別の分析 |
| アクション2 | integer | 該当プレイのアクションID（補助アクション1） | プレー種別の分析 |
| アクション3 | integer | 該当プレイのアクションID（補助アクション2） | プレー種別の分析 |

**主要アクションID例**:
- 1: 3Pシュート○
- 2: 3Pシュート×
- 3-6: 2Pシュート（アウトサイド/インサイドペイント、○/×）
- 7-8: フリースロー（○/×）
- 9: ディフェンスリバウンド
- 10: オフェンスリバウンド
- 11: ブロックショット
- 12: アシスト
- 13: ターンオーバー
- 14: スティール
- 29: ダンク
- 35: ファストブレイク（速攻）

#### 1.3 座標情報（座標付きイベントデータのみ）

| 項目名 | データ型 | 説明 | 活用可能性 |
|--------|----------|------|------------|
| エリア | integer | 該当アクションが発生したエリア（図1のエリア番号に紐付く） | **プレー位置の可視化** |
| X座標 | number | 該当アクションが発生したx座標（0～100、コート左下が(0,0)、右上が(100,100)） | **プレー位置の可視化** |
| Y座標 | number | 該当アクションが発生したy座標（0～100） | **プレー位置の可視化** |
| サイド | varchar | 該当アクションが発生した際のオフェンス側の攻撃方向（left/right、前半と後半で入れ替わる） | 攻撃方向の分析 |

**座標の仕様**:
- コートの左下を(x,y):(0,0)、右上を(x,y):(100,100)とする
- B.LEAGUEのコートの長さは28m、エンドラインから3PTラインのトップまでの長さは8.325m
- 例: 左コートの3PTラインのトップのx座標 = 8.325 / 28 * 100 = 29.7
- **注意**: 「エリア、x座標、y座標、サイド」は「アクション」がシュートの時のみデータが入る

**活用アイデア例**:
- シュート位置をコート上で可視化
- シュート成功率をエリア別に分析
- インパクトのあるシュートシーンの可視化
- 3Pシュートと2Pシュートの分布を可視化

## 💡 ペルソナ設定への活用

### ペルソナ例: Bリーグの観戦体験が無い人

**課題**:
- バスケットボールは大きなイベント（NBAなど）はテレビで見るが、Bリーグの生の観戦をするほど興味が湧かない
- 選手の特徴や魅力がわからない
- データを見ても面白さが伝わらない

**データ活用アイデア**:
1. **選手の特徴を数値で見える化**
   - 得点数、アシスト数、リバウンド数などの数値をインパクトのあるビジュアルで表現
   - ダンク、ブロックショットなどインパクトのあるプレーを可視化
   - シュート位置をコート上で可視化

2. **インパクトのある映像コンテンツ**
   - シュート位置をコート上で可視化した映像
   - ダンク、ブロックショットなどのハイライトシーン
   - 選手の動きを数値で見える化

3. **マーケティング施策**
   - 観客動員データから人気カードを分析
   - 時間帯別の観客動員分析
   - イベントや飲食店舗との連携企画
   - スポーツバーでのリアルタイムデータ可視化

4. **コンテンツ生成**
   - 選手の成績を自動で可視化したコンテンツ
   - インパクトのあるプレーシーンの自動抽出
   - SNS投稿自動生成
