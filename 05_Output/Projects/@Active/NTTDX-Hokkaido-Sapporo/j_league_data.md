# 2_トラッキングデータ_サッカー

```
| 項目名    | 項目説明・定義                                                                                                                         |
|:----------|:---------------------------------------------------------------------------------------------------------------------------------------|
| GameID    | 試合を一意に特定するID                                                                                                                 |
|           | イベントデータの「試合ID」と同じ値が入る                                                                                               |
| Frame     | トラッキングシステムのフレーム番号                                                                                                     |
|           | 秒間25フレーム（あるフレームから25進んだフレームが1秒後のフレーム）                                                                    |
|           | これをキーにして結合することでイベントデータと紐づけ可能                                                                               |
| HA        | ホーム/アウェイ特定用のフラグ                                                                                                          |
|           | 1がホーム、2がアウェイ、0はボール                                                                                                      |
| SysTarget | （トラッキングシステム内の物体検出番号のため、使用不可）                                                                               |
| No        | 選手の背番号、ボールの場合は0                                                                                                          |
| X         | ピッチ上のX座標で-5250～＋5250 の値をとる                                                                                              |
|           | ピッチサイズ105m x 68m の105m側                                                                                                        |
|           | 座標位置の詳細は下図を参照                                                                                                             |
| Y         | ピッチ上のY座標で-3400～＋3400 の値をとる                                                                                              |
|           | ピッチサイズ105m x 68m の68m側                                                                                                         |
|           | 座標位置の詳細は下図を参照                                                                                                             |
| Speed     | トラッキングシステム内で算出された移動速度、単位はkm/h                                                                                 |
|           | （例）29.3＝29.3km/h                                                                                                                   |
|           | ※ボールのXとYについて                                                                                                                  |
|           | イベントデータが発生したタイミングの選手のトラッキングデータ座標から計算した推定位置であり、トラッキングシステムが捕捉した位置ではない |
|           | 例1：A選手パス⇒B選手トラップの場合、A選手のトラッキングデータ座標からB選手のトラッキングデータ座標への等速直線運動とする               |
|           | 例2：シュート⇒ボールアウトのようにラインを割った場合は、A選手のトラッキングデータ座標からラインを割った位置への等速直線運動とする      |
|           | 例3：C選手ドリブルの場合、ドリブル開始から終了までボールの座標はC選手のトラッキングデータ座標とする                                    |
|           | ドリブル以外でボールを保持したまま動いている場合も同様                                                                                 |
|           | インプレーでない（ボールが外に出ている）間のボールの推定位置データは含まれない                                                         |
|           | トラッキングデータの座標位置                                                                                                           |
|           | （参考）イベントデータの座標位置　　※Y軸の正負はトラッキングデータと逆であることに注意                                                 |
```
