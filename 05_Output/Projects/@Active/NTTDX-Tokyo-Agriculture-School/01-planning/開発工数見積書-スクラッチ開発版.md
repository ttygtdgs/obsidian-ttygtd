## ⚠️ 前提条件・制約事項

### 技術スタック
- **Frontend**: Next.js 15, TypeScript, Tailwind CSS
- **Backend**: Node.js/Express, PostgreSQL, Prisma ORM
- **Infra**: AWS (ECS, RDS, CloudFront, Route53)
- **Auth**: AWS Cognito
- **Security**: HTTPS, CORS, CSRF対策

### 除外事項
- AI機能の組み込み（今回は不使用）
- iOS/Androidアプリ開発
- 運用・保守サービス
- ユーザー教育・トレーニング

### リスク要因
- **要件変更**: 仕様変更時は別途見積もり
- **外部API連携**: 必要な場合、追加工数発生
- **セキュリティ要件**: 学校向けのため、高度なセキュリティ対応
