# 構築見積依頼書

**案件名**: 東京都農業高校向け補助金支給事務システム構築
**依頼日**: 2026年2月14日
**回答期限**: ____年__月__日
**依頼元**: DXPN SD事業部

---

## 1. 依頼概要

東京都の農業高校（対象校複数）に通う高校生向けの**資格取得補助金の支給事務を管理するWebシステム**の構築を依頼したい。

既存システム（R7年度に日本ビジネスクリエイトが1,245万円で受注・運用中）があり、R8年度以降の運用・改善を想定している。**既存システムの改修**または**スクラッチでの新規開発**のいずれかで見積もりをお願いしたい。

### 座組

| 役割 | 担当 |
|------|------|
| フロント・入札対応・PM | DXPN 教育事業部 |
| BPO業務（問合せ対応・申請受付等） | ネクシア |
| システム構築・AI機能開発 | **外部パートナー（本依頼先）** |

---

## 2. 要望・スコープ

### 2-1. 基本機能（必須）

**ユーザー側（高校生・保護者）**
- アカウント登録・ログイン
- 補助金交付申請の提出
- 情報変更申請の提出
- 実績報告・証明書類の提出（資格受験後）
- 支払申請の提出（額の決定通知後）
- 申請ステータスの確認

**管理者側（事務局）**
- ユーザー（高校生）管理
- 申請レビュー（承認・否認）
- 申請データのCSV/Excel出力

### 2-2. AI活用機能（オプション・別見積可）

以下のAI機能について、個別またはセットでの見積もりを希望する。

| # | 機能 | 概要 |
|---|------|------|
| 1 | FAQチャットボット | 生徒・保護者からの問い合わせ自動応答（24時間対応） |
| 2 | 学生証画像確認 | OCRによる学生証の自動読み取り・本人確認支援 |
| 3 | 領収書画像確認 | 受験料領収書の自動読み取り・金額整合性チェック |
| 4 | 口座情報確認 | 振込先口座情報の自動読み取り・整合性チェック |

### 2-3. 非機能要件

| 項目 | 要件 |
|------|------|
| 可用性 | 稼働率 99.5%以上 |
| レスポンス | ページ表示 2秒以内 |
| セキュリティ | 教育機関向けセキュリティ基準準拠、個人情報保護 |
| モバイル対応 | スマートフォン・タブレット対応（レスポンシブ） |
| ファイル処理 | PDF・画像ファイルのアップロード・表示 |

## 6. 補足情報

### 技術スタック（参考・スクラッチ開発の場合）

社内で想定しているスタック。変更提案があれば歓迎する。

- **Frontend**: Next.js + TypeScript + Tailwind CSS
- **Backend**: Node.js + Express + PostgreSQL + Prisma
- **Infrastructure**: AWS（ECS, RDS, CloudFront, Cognito）
- **DevOps**: GitHub Actions, Docker

### 現状の業務フロー

```
生徒 → アカウント登録 → 補助金交付申請 → [変更申請] → 実績報告 → 額の決定 → 支払申請 → 振込
                                    ↕
                          管理者: 申請レビュー（承認/否認）
```

### 既存システム情報

- R7年度に日本ビジネスクリエイトが構築・運用
- R8年度の要件はR7年度をほぼ踏襲
- 既存システムの技術スタック詳細は現時点で未把握（今後確認予定）
