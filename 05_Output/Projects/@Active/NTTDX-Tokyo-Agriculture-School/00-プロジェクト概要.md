## 🎯 プロジェクト概要

東京都農業高校向けのデジタルプラットフォームをスクラッチ開発するプロジェクト。

### 目的
- 農業高校生の学習・実習管理をデジタル化
- 教師の指導業務効率化
- 生徒の自己学習促進

### 対象ユーザー
- **生徒**: 農業高校生（1-3年生）
- **教師**: 農業科教師
- **管理者**: 学校管理者

---

## 💰 予算・見積もり

- **見積総額**: 5,600,000円（税別）
- **開発期間**: 3ヶ月
- **体制**: 実装担当者1名 + レビュアー1名

[詳細見積書](./01-planning/開発工数見積書-スクラッチ開発版.md)

---

## 📋 主要要件

### 機能要件
1. **学習管理機能**: カリキュラム管理、進捗追跡
2. **実習記録機能**: 農業実習の記録・管理
3. **ポートフォリオ機能**: 生徒の成果物管理
4. **教師管理機能**: 指導記録、評価管理

### 非機能要件
- **セキュリティ**: 学校向けセキュリティ基準
- **可用性**: 99.5%以上の稼働率
- **レスポンス**: ページ表示2秒以内
- **モバイル対応**: スマートフォン対応

---

## 🛠️ 技術スタック

- **Frontend**: Next.js 15 + TypeScript + Tailwind CSS
- **Backend**: Node.js + Express + PostgreSQL + Prisma
- **Infra**: AWS (ECS, RDS, CloudFront, Cognito)
- **DevOps**: GitHub Actions, Docker
