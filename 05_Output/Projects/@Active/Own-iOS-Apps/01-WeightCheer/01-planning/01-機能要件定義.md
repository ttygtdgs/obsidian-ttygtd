# WeightCheer - 機能要件定義

## 1. ユーザーロール

### 記録者（メインユーザー）
- 体重を記録する人（例：夫）
- 1グループにつき1名

### サポーター（応援ユーザー）
- 記録者を応援する人（例：妻、友人）
- 1グループにつき複数名可

## 2. 機能一覧

### 2.1 認証・アカウント

| ID | 機能 | 説明 | 優先度 |
|----|------|------|--------|
| AUTH-01 | Apple Sign In | Apple IDでログイン | 必須 |
| AUTH-02 | プロフィール設定 | 名前・アイコンの設定 | 必須 |
| AUTH-03 | アカウント削除 | データ含む完全削除 | 必須 |

### 2.2 グループ管理

| ID | 機能 | 説明 | 優先度 |
|----|------|------|--------|
| GRP-01 | グループ作成 | 記録者がグループを作成 | 必須 |
| GRP-02 | 招待リンク生成 | サポーターを招待するリンク発行 | 必須 |
| GRP-03 | グループ参加 | 招待リンクからグループに参加 | 必須 |
| GRP-04 | メンバー一覧 | グループメンバーの確認 | 必須 |
| GRP-05 | メンバー削除 | 記録者がメンバーを削除 | あると良い |

### 2.3 体重記録

| ID | 機能 | 説明 | 優先度 |
|----|------|------|--------|
| WGT-01 | 体重入力 | 日付と体重（0.1kg単位）を記録 | 必須 |
| WGT-02 | 体重編集 | 過去の記録を修正 | 必須 |
| WGT-03 | 体重削除 | 誤った記録を削除 | 必須 |
| WGT-04 | 目標体重設定 | 目標値を設定 | 必須 |
| WGT-05 | 一言メモ | 記録時にメモを添付（例：「今日は飲み会」） | あると良い |
| WGT-06 | 体重グラフ | 日次・週次・月次のグラフ表示 | 必須 |
| WGT-07 | 統計表示 | 平均・最高・最低・変動幅の表示 | あると良い |

### 2.4 応援・コメント

| ID | 機能 | 説明 | 優先度 |
|----|------|------|--------|
| CHR-01 | コメント投稿 | 記録に対してテキストコメント | 必須 |
| CHR-02 | スタンプ送信 | ワンタップで応援スタンプを送信 | 必須 |
| CHR-03 | コメント一覧 | 記録ごとのコメント表示 | 必須 |
| CHR-04 | コメント削除 | 自分のコメントを削除 | 必須 |

### 2.5 通知

| ID | 機能 | 説明 | 優先度 |
|----|------|------|--------|
| NTF-01 | 記録通知 | 記録者が記録したらサポーターに通知 | 必須 |
| NTF-02 | コメント通知 | コメントが付いたら記録者に通知 | 必須 |
| NTF-03 | リマインダー | 未記録時にリマインド（時間設定可） | あると良い |
| NTF-04 | マイルストーン通知 | 目標達成・連続記録などの通知 | あると良い |

### 2.6 その他

| ID | 機能 | 説明 | 優先度 |
|----|------|------|--------|
| ETC-01 | 連続記録カウント | 何日連続で記録しているか表示 | あると良い |
| ETC-02 | 達成バッジ | マイルストーンに応じたバッジ付与 | 将来検討 |
| ETC-03 | データエクスポート | CSV形式でのデータ出力 | 将来検討 |
| ETC-04 | HealthKit連携 | Apple ヘルスケアとの同期 | 将来検討 |

## 3. 非機能要件

| 項目 | 要件 |
|------|------|
| 対応OS | iOS 17以上 |
| 対応端末 | iPhone |
| 言語 | 日本語（初期リリース） |
| データ保持 | 無期限 |
| オフライン | オフライン時も記録可能、復帰後に同期 |
| セキュリティ | 体重データは暗号化して保存 |
| プライバシー | 体重はグループメンバーのみ閲覧可 |

## 4. 画面遷移概要

```
ログイン → ホーム（体重記録一覧）
              ├── 体重入力
              ├── グラフ表示
              ├── コメント詳細
              ├── グループ管理
              └── 設定
```
