# WeightCheer - 画面構成

## 画面遷移図

```
[スプラッシュ]
    │
    ▼
[ログイン画面]
    │ Apple Sign In
    ▼
[初回セットアップ] ※初回のみ
    │ プロフィール設定 → グループ作成 or 参加
    ▼
[タブバー]
    ├── [ホーム] ← メインタブ
    ├── [グラフ]
    └── [設定]
```

## 各画面の詳細

### 2. 初回セットアップ（ステップ形式）

**Step 1: プロフィール設定**
```
┌─────────────────────────┐
│ ← 　プロフィール設定     │
│─────────────────────────│
│                         │
│        [アイコン]        │
│       タップで変更       │
│                         │
│  名前                   │

│  │ たろう              │  │
│  └───────────────────┘  │
│                         │
│                         │
│  ┌───────────────────┐  │
│  │     次へ →         │  │
│  └───────────────────┘  │
└─────────────────────────┘
```

**Step 2: ロール選択**
```
┌─────────────────────────┐
│ ← 　あなたの役割は？     │
│─────────────────────────│
│                         │
│  ┌───────────────────┐  │

│  │  体重を記録します   │  │
│  └───────────────────┘  │
│                         │

│  │  仲間を応援します   │  │
│  └───────────────────┘  │
│                         │
└─────────────────────────┘
```

**Step 2a: 記録する人 → グループ作成**
```
┌─────────────────────────┐
│ ← 　グループ作成         │
│─────────────────────────│
│                         │
│  グループ名              │
│  ┌───────────────────┐  │

│                         │
│  ┌───────────────────┐  │
│  │   グループを作成    │  │
│  └───────────────────┘  │
└─────────────────────────┘
```

**Step 2b: 応援する人 → 招待リンク入力**
```
┌─────────────────────────┐
│ ← 　グループに参加       │
│─────────────────────────│
│                         │
│  招待リンクを受け取って  │
│  グループに参加しましょう│
│                         │
│  ┌───────────────────┐  │
│  │ リンクを貼り付け    │  │

│                         │

│  │     参加する        │  │
│  └───────────────────┘  │
└─────────────────────────┘
```
