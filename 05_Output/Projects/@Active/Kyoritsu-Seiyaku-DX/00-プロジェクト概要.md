# 🎯 Project: 共立製薬-スコアシート自動化

## 📌 Executive Summary

**一行説明**: 監視員が紙のスコアシートに記入した内容を、PDF化してExcelに転記する業務フローを**半自動化**するシステム開発プロジェクト

**プロジェクトの目的**: 
- 紙からExcelへの手入力転記作業を効率化（完全自動化は目標としない）
- 既存の紙ベースワークフローを維持しつつ、データ化を半自動化
- **誤転記の混入を防止**し、研究資料として利用可能な品質を確保
- confidence（信頼度）を活用した効率的な人手確認フロー

**想定成果**: 
- PDFからExcelへの半自動データ抽出システム（OCR + 人手確認）
- Google Document AI Custom Extractorによる高精度データ抽出
- confidenceを活用した確認工数の削減（全体の20-30%以下）
- 月間200-300件の処理に対応可能なシステム

## 🎯 Goals & Success Metrics

### Primary Goals
1. PDFからデータ抽出精度: 85%以上（信頼度スコア）
2. 処理時間: 1件あたり30秒以内
3. Excel出力フォーマット: 既存フォーマット100%準拠
4. 転記作業時間: 80%削減（10分→2分）

### Success Criteria
- [ ] PoC完了（Custom Extractor学習完了、confidence分析完了）
- [ ] 人手確認件数が全体の20-30%以下に削減
- [ ] 誤転記が研究資料として問題にならない水準で抑制
- [ ] プロトタイプシステム完成
- [ ] 本番環境稼働開始
- [ ] ユーザー教育完了（スキャン品質、帳票記入ルール含む）
