# 提案1: スコアシートDX化

## 実施イメージ

**現状**
```
紙スコアシート → 手作業でExcel転記
```

**提案**
```
紙スコアシート → OCR自動読取 → 確認・修正 → Excel出力
```

**ポイント**
- OCRで自動読取、低信頼度の項目のみ人が確認
- 転記作業80%削減を目指す

---

## 進め方

**2段階アプローチ**

```
Phase 1: PoC（2ヶ月）
・OCR精度検証
・デモシステム構築
・Go/No-Go判断

    ↓ 精度85%以上でGo

Phase 2: 本開発（3-4ヶ月）
・本番システム構築
・運用移行
```

**Go/No-Go判断基準**
- OCR精度: 85%以上
- 人手確認対象: 30%以下

---

## スケジュール

| フェーズ | 期間 | 内容 |
|---------|------|------|
| PoC | 2ヶ月 | 精度検証、デモ構築 |
| 判断 | - | Go/No-Go判断 |
| 本開発 | 3-4ヶ月 | 本番システム構築 |

**費用**

# 提案2: データ基盤構築

## 概要

**目的**
- Excelデータを構造化データとして格納

**構成**
```
Dropbox(Excel)
    ↓
インポートスクリプト
    ↓
Snowflake（構造化データ）
    ↓
BIツール / 検索UI
```
