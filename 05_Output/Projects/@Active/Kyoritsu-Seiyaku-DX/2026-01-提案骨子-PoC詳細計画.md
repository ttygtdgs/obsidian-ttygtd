### 4.2 開発内容

#### A. Excelインポートスクリプト

| 機能 | 内容 |
|------|------|
| **Excelパーサー** | 既存Excelフォーマットを解析し、データを抽出 |
| **データ変換** | 抽出データを構造化フォーマットに変換 |
| **バリデーション** | 必須項目チェック、データ型検証、値域チェック |
| **重複チェック** | 日付+動物ID等をキーに重複データを検知 |
| **エラーハンドリング** | パース失敗、検証エラー時の報告・スキップ処理 |

```python
# イメージ: Excelインポートスクリプトの処理フロー
def import_excel(file_path):
    # 1. Excelファイル読み込み
    df = parse_excel(file_path)

    # 2. データ変換・正規化
    records = transform_to_records(df)

    # 3. バリデーション
    valid_records, errors = validate_records(records)

    # 4. 重複チェック
    new_records = filter_duplicates(valid_records)

    # 5. データ基盤に登録
    save_to_database(new_records)

    # 6. 結果レポート
    return ImportResult(
        success=len(new_records),
        duplicates=len(valid_records) - len(new_records),
        errors=errors
    )
```
