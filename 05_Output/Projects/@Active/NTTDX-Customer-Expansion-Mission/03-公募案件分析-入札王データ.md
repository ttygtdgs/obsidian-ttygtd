## 付録：データ処理の注記

### クレンジング（2,507件→2,051件、456件除去）

| ノイズカテゴリ | 件数 | 除去理由 |
|-------------|:---:|---------|
| 防衛省部品（非AI） | ~170件 | DIAPHRAGM, DRAG LINK等の航空機部品 |
| 空港RAG無線設備 | ~200件 | RAG/ＲＡＧ = Remote Air-Ground（航空無線）。全角ＲＡＧ含む |
| 和歌山県閲覧表 | ~30件 | Nextorage, Orage, Crag等のブランド名 |
| 書籍・試薬・消耗品 | ~30件 | AI関連書籍購入、医療試薬、保護フィルム等 |
| HW購入（非AI） | ~10件 | iPad、サーバ、ストレージ等のハードウェア購入 |
| その他 | ~16件 | 建築工事、JICA海外案件等 |

### 重複統合（2,051件→1,269件、782件統合）

同一案件が「入札中（N）」と「落札結果（R）」の2レコードで登録されるケースを統合。件名＋発注機関の正規化マッチで統合。R（金額情報あり）を優先、N（日付情報あり）で年度判定。

### フェーズ分類ロジック（v2）

1. **①研修**: 研修/セミナー/ワークショップ/職業訓練/講座 を含む
2. **②設計**: ロードマップ/計画策定/RFI/情報提供依頼/意見招請 を含む
3. **③PoC**: 実証/検証/試行/トライアル/パイロット を含む
4. **⑤運用**: 運用/保守/ライセンス/サービス利用/利用契約/賃貸借 を含む
5. **④構築**: 上記いずれにも該当しない案件（デフォルト）
