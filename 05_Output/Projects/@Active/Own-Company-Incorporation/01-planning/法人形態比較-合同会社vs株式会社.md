# 法人形態比較: 合同会社（LLC） vs 株式会社（Corp）

## 概要

法人設立にあたり、日本で最も一般的な2つの法人形態を多角的に比較する。
個人事業主からの法人化において、どちらが自身のユースケースに最適かを判断するための資料。

> ※要専門家確認: 本資料の法務・税務情報は一般的な情報に基づくものであり、個別の状況に応じて専門家に確認してください。

---

## 比較マトリクス

| 比較項目 | 合同会社（LLC） | 株式会社（Corp） |
|---------|----------------|-----------------|
| **設立費用** | 約6〜10万円（定款認証不要） | 約20〜25万円（定款認証+登録免許税） |
| **登録免許税** | 6万円（最低額） | 15万円（最低額） |
| **定款認証** | 不要 | 必要（公証人手数料3〜5万円） |
| **設立期間** | 1〜2週間 | 2〜3週間 |
| **代表者肩書** | 代表社員 | 代表取締役 |
| **社会的信用度** | やや低い（認知度が低い） | 高い（一般に広く認知） |
| **意思決定速度** | 速い（社員の合意で即決） | 取締役会・株主総会の手続きが必要 |
| **出資と経営** | 出資者＝経営者（分離不可） | 出資者と経営者を分離可能 |
| **利益配分** | 出資比率によらず自由に設定可能 | 出資比率に応じた配当 |
| **資金調達** | 社債のみ（株式発行不可） | 株式発行・社債いずれも可能 |
| **上場** | 不可 | 可能 |
| **決算公告義務** | なし | あり（官報掲載費 約7.5万円/年） |
| **役員任期** | なし（無期限） | 最長10年（変更登記が必要） |
| **定款変更** | 社員の同意で可能 | 株主総会の特別決議が必要 |
| **組織変更** | 株式会社への変更可能 | 合同会社への変更可能 |
| **法人税率** | 同一 | 同一 |
| **ランニングコスト** | 低い（決算公告・役員変更登記不要） | やや高い（決算公告・役員変更登記費用） |

### 2. 信用度・ブランド力

- **株式会社**: 「株式会社」は日本で最も認知度の高い法人形態。BtoB取引・銀行融資・採用で有利に働くことが多い
- **合同会社**: 認知度は上昇中だが、まだ「株式会社」ほどのブランド力はない。ただしApple Japan、Google Japan、Amazon Japanなど大手外資系の日本法人が合同会社を採用している実績あり

### 3. 意思決定と運営の柔軟性

- **合同会社**: 所有と経営が一致しているため、迅速な意思決定が可能。定款自治の範囲が広く、柔軟な運営が可能
- **株式会社**: 会社法に基づく組織運営（株主総会、取締役会等）が必要。1人会社であれば実質的に合同会社と同程度の柔軟性

### 4. 将来の拡張性

- **株式会社**: 株式発行による資金調達、IPO（上場）が可能。ストックオプション制度も活用可能
- **合同会社**: 株式発行不可のため、外部からの出資受入に制限あり。ただし、株式会社への組織変更は可能（費用・手間はかかる）

## ユースケース別推奨

### 合同会社が向いているケース

1. **フリーランス・個人事業主の法人化**
   - 1人または少人数での運営
   - 設立コストを抑えたい
   - 迅速な意思決定を重視

2. **副業の法人化**
   - 節税が主目的
   - 外部からの信用度はそこまで重要でない
   - 小規模な運営

3. **コンサルティング・IT受託**
   - サービス提供型で株式発行不要
   - ランニングコストを最小化したい
   - Apple Japan等の前例あり

4. **不動産管理法人**
   - 資産管理が主目的
   - 対外的な信用度が重要でない

### 株式会社が向いているケース

1. **BtoB営業が主軸の事業**
   - 大企業との取引が多い
   - 名刺交換・信用調査で「株式会社」が有利

2. **将来の資金調達を想定**
   - VCからの出資受入の可能性がある
   - IPOを視野に入れている

3. **採用を積極的に行う事業**
   - 求職者にとっての信頼感
   - ストックオプション制度の活用

4. **金融機関からの融資を重視**
   - 銀行融資の審査で有利になる場合がある

## 判断フローチャート

```
外部から資金調達（VC・IPO）の予定がある？
├─ YES → 株式会社
└─ NO
   └─ BtoB大企業取引がメインか？
      ├─ YES → 株式会社
      └─ NO
         └─ 従業員を積極採用する予定か？
            ├─ YES → 株式会社（ストックオプション活用）
            └─ NO
               └─ 設立コスト・ランニングコストを最小化したいか？
                  ├─ YES → 合同会社
                  └─ NO → 株式会社（ブランド力優先）
```

## 結論・推奨

### 個人事業主・フリーランスからの法人化の場合

**推奨: 合同会社**

理由:
1. 設立費用が約10〜15万円安い
2. 決算公告義務がなく、ランニングコストが低い
3. 役員任期の制限がなく、変更登記の手間・費用が不要
4. 1人運営で迅速な意思決定が可能
5. 将来的に株式会社へ組織変更も可能

ただし以下の条件に当てはまる場合は株式会社を検討:
- 大企業との取引で「株式会社」のブランドが必要
- 2〜3年以内に外部資金調達を予定
- 積極的な採用計画がある

> ※要専門家確認: 最終的な法人形態の選択は、税理士・司法書士と相談の上、自身の事業計画に基づいて決定してください。
