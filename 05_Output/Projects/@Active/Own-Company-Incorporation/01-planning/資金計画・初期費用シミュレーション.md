## 2. 資本金の設定

### 資本金額の考え方（※要専門家確認）

| 資本金額 | メリット | デメリット |
|---------|---------|----------|
| **1円〜** | 法律上は1円から設立可能 | 信用度が極めて低い |
| **10〜50万円** | 設立費用を最小化 | 法人口座開設が難しい場合あり |
| **100万円** | バランスが良い、口座開設もしやすい | 定款認証手数料が4万円に上がる（株式会社） |
| **300万円** | 一定の信用力、融資審査でも安心 | 定款認証手数料が5万円に上がる（株式会社） |
| **1,000万円以上** | 高い信用力 | 初年度から消費税課税事業者になる |

### 推奨: 50〜100万円
- 法人口座開設のハードルをクリアしやすい
- 消費税の免税事業者として最大2年間免除の恩恵を受けられる（※要専門家確認）
- 登録免許税が最低額のまま

## 5. 法人化の損益分岐点（※要専門家確認）

| 項目 | 個人事業 | 法人（役員報酬600万円） |

| 所得税+住民税 | 約130万円 | 約80万円（給与所得控除後） |
| 法人税等 | — | 約15万円（法人所得100万円に対して） |
| 社会保険料差額 | — | +20万円程度（厚生年金分） |
| **概算税負担合計** | **約130万円** | **約115万円** |
| **差額（節税効果）** | | **約15万円/年** |

> この試算は非常に簡略化したものです。実際の節税効果は、役員報酬の設定、各種控除、事業経費の構成等により大きく変わります。必ず税理士に相談してください。※要専門家確認
