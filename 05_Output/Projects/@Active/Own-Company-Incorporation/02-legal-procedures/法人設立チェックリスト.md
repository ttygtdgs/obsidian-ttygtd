# 法人設立チェックリスト

法人設立の全手続きをPhase A〜Eに分けたチェックリスト。合同会社（LLC）と株式会社（Corp）の両方に対応。

> ※要専門家確認: 法務手続きの詳細は管轄の法務局・公証役場に確認してください。司法書士への依頼を推奨します。

---

## Phase A: 設立準備（1〜2週間）

### 基本事項の決定
- [ ] **法人形態の決定**: 合同会社 or 株式会社 → [[法人形態比較-合同会社vs株式会社]]
- [ ] **商号（会社名）の決定**
  - 同一住所に同一商号がないか確認（法務局で調査 or オンライン検索）
  - 「合同会社○○」or「株式会社○○」の形式
  - 使用可能文字: 漢字、ひらがな、カタカナ、ローマ字、アラビア数字、一部の記号
- [ ] **事業目的の決定**
  - 定款に記載する事業目的リストを作成
  - 将来行う可能性のある事業も含める
  - 許認可が必要な業種は正確な表現が必要 ※要専門家確認
- [ ] **本店所在地の決定** → [[バーチャルオフィス・住所検討]]
  - 自宅 / バーチャルオフィス / レンタルオフィス
  - 賃貸の場合、法人登記可能か管理会社に確認
- [ ] **資本金額の決定** → [[資金計画・初期費用シミュレーション]]
  - 推奨: 50〜100万円
  - 1,000万円未満にすること（消費税免税のため）
- [ ] **決算期（事業年度）の決定** → [[決算期・届出スケジュール]]
  - 推奨: 設立月の前月
- [ ] **役員（社員/取締役）の決定**
  - 合同会社: 社員（出資者＝経営者）
  - 株式会社: 取締役（1人以上）、場合により監査役
- [ ] **役員報酬の決定**（概算でOK、設立後3ヶ月以内に正式決定）

### 準備物の手配
- [ ] **法人印鑑の発注**（届くまで3〜7日程度）
  - 代表印（実印）: 18mm丸印
  - 銀行印: 16.5mm丸印
  - 角印（社印）: 24mm角印
  - ゴム印（住所印）: 任意
- [ ] **発起人（出資者）の印鑑証明書取得**
  - 市区町村役場で取得（マイナンバーカードがあればコンビニ発行可）
  - 3ヶ月以内のもの
- [ ] **個人の実印の準備**（未登録なら印鑑登録）

---

## Phase B: 定款作成・認証（1週間）

### 定款の作成
- [ ] **定款の起案**
  - 必要的記載事項:
    - 商号
    - 事業目的
    - 本店所在地
    - 設立に際して出資される財産の額または最低額
    - 発起人の氏名・住所
    - （株式会社）発行可能株式総数
  - 任意的記載事項:
    - 事業年度
    - 役員の人数・任期
    - 株式の譲渡制限（株式会社）
- [ ] **定款の内容確認**（司法書士依頼の場合はドラフトの確認）
- [ ] **電子定款の作成**（印紙代4万円の節約のため電子定款を推奨）

### 定款認証（株式会社のみ）
- [ ] **公証役場での認証予約**（本店所在地の都道府県内の公証役場）
- [ ] **定款認証の実施**
  - 持参物: 定款3通、印鑑証明書、発起人の実印、身分証明書、認証手数料
- [ ] **認証済み定款の受領**

### 合同会社の場合
- [ ] 定款認証は不要（社員全員が署名または記名押印で成立）

---

## Phase C: 登記申請（1〜2週間）

### 資本金の払い込み
- [ ] **資本金の払い込み**
  - 発起人の個人口座に資本金額を振り込み
  - 振り込み日は定款認証日以降（株式会社）、定款作成日以降（合同会社）
- [ ] **払い込み証明書の作成**
  - 通帳のコピー（表紙・1ページ目・該当ページ）
  - ネット銀行の場合はWeb明細の印刷

### 登記申請
- [ ] **登記申請書の作成**
- [ ] **必要書類の準備** → [[必要書類一覧]]
- [ ] **法務局への登記申請**
  - 窓口持参 / 郵送 / オンライン（登記・供託オンライン申請システム）
  - 登録免許税の納付（収入印紙 or 電子納付）
  - **申請日 = 会社設立日** となる
- [ ] **登記完了の確認**（申請から約1〜2週間）
- [ ] **登記事項証明書の取得**（3〜5通程度取得しておく）
- [ ] **法人番号の確認**（国税庁法人番号公表サイトで確認可能）
- [ ] **印鑑カードの取得**（法務局窓口で申請）

---

## Phase D: 設立後届出（2〜4週間）

### 税務関係
- [ ] **税務署への届出** → [[税務・社会保険手続き]]
  - [ ] 法人設立届出書（設立から2ヶ月以内）
  - [ ] 青色申告の承認申請書（設立から3ヶ月以内 or 第1期末のいずれか早い日の前日）**← 必須**
  - [ ] 給与支払事務所等の開設届出書（開設から1ヶ月以内）
  - [ ] 源泉所得税の納期の特例の承認申請書（随時、早めに）
- [ ] **都道府県税事務所への届出**
  - [ ] 法人設立届出書（15日〜1ヶ月以内、自治体による）
- [ ] **市区町村への届出**
  - [ ] 法人設立届出書（1〜2ヶ月以内、自治体による）

### 社会保険関係
- [ ] **年金事務所への届出**（設立から5日以内）→ [[税務・社会保険手続き]]
  - [ ] 健康保険・厚生年金保険 新規適用届
  - [ ] 被保険者資格取得届
  - [ ] 被扶養者（異動）届（該当者がいる場合）

### 労働保険関係（従業員を雇用する場合）
- [ ] **労働基準監督署**
  - [ ] 労働保険関係成立届（適用事業になった日の翌日から10日以内）
  - [ ] 労働保険概算保険料申告書（成立日から50日以内）
- [ ] **ハローワーク**
  - [ ] 雇用保険適用事業所設置届（設置日の翌日から10日以内）
  - [ ] 雇用保険被保険者資格取得届（雇用日の翌月10日まで）

### 金融関係
- [ ] **法人口座の開設**
  - メガバンク / ネット銀行 / 信用金庫
  - 必要書類: 登記事項証明書、定款の写し、代表者の本人確認書類、法人印鑑等
  - 審査に1〜3週間かかることがある

---

## Phase E: 運営開始準備

### 経理・バックオフィス
- [ ] **会計ソフトの導入・初期設定**
  - freee / マネーフォワード / 弥生 等
  - 法人情報・勘定科目・決算期の設定
- [ ] **税理士との顧問契約**（推奨）
- [ ] **請求書テンプレートの作成**（インボイス制度対応）
- [ ] **法人クレジットカードの申し込み**（任意）
- [ ] **経費精算ルールの策定**

### 事業開始
- [ ] **名刺の作成**
- [ ] **法人メールアドレスの取得**
- [ ] **Webサイト・SNSの更新**（法人名への変更等）
- [ ] **既存取引先への法人化通知**
- [ ] **契約書の法人名義への切り替え**
- [ ] **インボイス登録番号の取得**（課税事業者の場合）※要専門家確認

### 許認可（該当する場合のみ）
- [ ] **業種別許認可の申請** ※要専門家確認
  - 例: 人材派遣業（厚生労働大臣の許可）
  - 例: 古物商（公安委員会の許可）
  - 例: 飲食業（保健所の営業許可）
