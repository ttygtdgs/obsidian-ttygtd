# 税務・社会保険手続き

法人設立後に必要な税務署・都道府県税事務所・市区町村・年金事務所への届出内容と期限を詳細に整理する。

> ※要専門家確認: 税務・社会保険の手続きは法改正や個別事情により変わる場合があります。税理士・社会保険労務士に相談してください。

---

## 1. 税務署への届出

### 1-1. 法人設立届出書

| 項目 | 内容 |
|------|------|
| **届出先** | 本店所在地を管轄する税務署 |
| **期限** | 設立の日から2ヶ月以内 |
| **届出書類** | 法人設立届出書（様式は国税庁Webサイトからダウンロード可能） |
| **添付書類** | 定款の写し、登記事項証明書、設立時の貸借対照表、株主名簿（出資者名簿） |
| **届出方法** | 窓口持参 / 郵送 / e-Tax（電子申告） |

**記載内容**:
- 法人名、本店所在地、代表者氏名
- 設立年月日、事業年度（決算期）
- 事業の目的
- 設立時の資本金額
- 届出の届出人（代表者）

### 1-2. 青色申告の承認申請書

| 項目 | 内容 |
|------|------|
| **届出先** | 本店所在地を管轄する税務署 |
| **期限** | **設立の日から3ヶ月以内、または第1期の事業年度終了の日のいずれか早い日の前日まで** |
| **届出書類** | 青色申告の承認申請書 |
| **添付書類** | なし |

**青色申告のメリット**（※要専門家確認）:
- **欠損金の繰越控除**: 最大10年間、赤字を翌期以降に繰り越して課税所得と相殺できる
- **少額減価償却資産の特例**: 30万円未満の資産を即時経費化（年間合計300万円まで）
- **各種特別控除・税額控除**: 中小企業投資促進税制等の適用

> **重要**: 青色申告の申請を忘れると、上記のメリットが第1期から受けられません。**設立後最優先で提出**してください。

### 1-3. 給与支払事務所等の開設届出書

| 項目 | 内容 |
|------|------|
| **届出先** | 本店所在地を管轄する税務署 |
| **期限** | 事務所開設から1ヶ月以内 |
| **届出書類** | 給与支払事務所等の開設届出書 |
| **添付書類** | なし |

**備考**: 役員報酬を支払う場合も「給与」に該当するため、1人法人でも必要。

### 1-4. 源泉所得税の納期の特例の承認申請書

| 項目 | 内容 |
|------|------|
| **届出先** | 本店所在地を管轄する税務署 |
| **期限** | 随時（早めに提出が望ましい） |
| **届出書類** | 源泉所得税の納期の特例の承認申請書 |
| **添付書類** | なし |
| **適用条件** | 給与支給人員が常時10人未満 |

**メリット**:
- 通常: 源泉所得税は毎月翌月10日までに納付
- 特例適用後: 年2回（1〜6月分を7月10日、7〜12月分を翌年1月20日）にまとめて納付
- 事務負担の大幅削減

### 1-5. その他の届出（必要に応じて）

| 届出書類 | 期限 | 備考 |
|---------|------|------|
| 棚卸資産の評価方法の届出書 | 第1期の確定申告書提出期限まで | 届出がなければ最終仕入原価法 |
| 減価償却資産の償却方法の届出書 | 第1期の確定申告書提出期限まで | 届出がなければ定額法 |
| 消費税課税事業者選択届出書 | 適用を受けようとする課税期間の初日の前日まで | 免税事業者があえて課税事業者を選択する場合 |
| 消費税簡易課税制度選択届出書 | 適用を受けようとする課税期間の初日の前日まで | 基準期間の売上が5,000万円以下の場合 |
| 適格請求書発行事業者の登録申請書 | 随時 | インボイス制度対応 ※要専門家確認 |

## 2. 都道府県税事務所への届出

### 法人設立届出書（都道府県）

| 項目 | 内容 |
|------|------|
| **届出先** | 本店所在地を管轄する都道府県税事務所 |
| **期限** | 設立から15日〜1ヶ月以内（都道府県により異なる）※要専門家確認 |
| **届出書類** | 法人設立届出書（都道府県の様式） |
| **添付書類** | 定款の写し、登記事項証明書 |

**主な都道府県別期限の例**:
| 都道府県 | 提出期限 |
|---------|---------|
| 東京都 | 設立から15日以内 |
| 神奈川県 | 設立から2ヶ月以内 |
| 大阪府 | 設立から2ヶ月以内 |

> ※要専門家確認: 管轄の都道府県税事務所に正確な期限を確認してください。

---

## 3. 市区町村への届出

### 法人設立届出書（市区町村）

| 項目 | 内容 |
|------|------|
| **届出先** | 本店所在地の市区町村（23区内は都税事務所で一括） |
| **期限** | 設立から1〜2ヶ月以内（自治体による） |
| **届出書類** | 法人設立届出書（市区町村の様式） |
| **添付書類** | 定款の写し、登記事項証明書 |

**注意**: 東京23区内に本店がある場合は、都税事務所への届出のみで市区町村への届出は不要。

## 4. 年金事務所への届出（社会保険）

### 概要
法人は役員1人（自分だけ）でも社会保険（健康保険・厚生年金保険）への加入が**法律上義務**。

### 4-1. 健康保険・厚生年金保険 新規適用届

| 項目 | 内容 |
|------|------|
| **届出先** | 本店所在地を管轄する年金事務所 |
| **期限** | 事実発生から5日以内 |
| **届出書類** | 健康保険・厚生年金保険 新規適用届 |
| **添付書類** | 登記事項証明書（発行から90日以内）、法人番号指定通知書のコピー |

### 4-2. 被保険者資格取得届

| 項目 | 内容 |
|------|------|
| **届出先** | 本店所在地を管轄する年金事務所 |
| **期限** | 事実発生から5日以内 |
| **届出書類** | 被保険者資格取得届 |
| **記載内容** | 被保険者の氏名、生年月日、マイナンバー、報酬月額、資格取得日等 |

### 4-3. 被扶養者（異動）届

| 項目 | 内容 |
|------|------|
| **届出先** | 本店所在地を管轄する年金事務所 |
| **期限** | 事実発生から5日以内 |
| **届出書類** | 被扶養者（異動）届 |
| **適用** | 扶養家族がいる場合のみ |
| **添付書類** | 収入確認書類等 |

### 社会保険料の概算（※要専門家確認）

役員報酬に基づく社会保険料の目安（2026年度の料率は変動する可能性あり）:

| 役員報酬（月額） | 健康保険料（会社負担） | 厚生年金（会社負担） | 合計（会社負担） |
|----------------|---------------------|-------------------|----------------|
| 150,000円 | 約7,500円 | 約14,000円 | 約21,500円 |
| 200,000円 | 約10,000円 | 約18,300円 | 約28,300円 |
| 300,000円 | 約15,000円 | 約27,450円 | 約42,450円 |
| 500,000円 | 約25,000円 | 約45,750円 | 約70,750円 |
| 800,000円 | 約40,000円 | 約59,475円 | 約99,475円 |

> 上記は標準報酬月額に基づく概算です。実際の保険料率は都道府県・年度により異なります。※要専門家確認

## 5. 労働保険（従業員を雇用する場合）

### 5-1. 労働基準監督署

| 届出書類 | 期限 | 備考 |
|---------|------|------|
| 労働保険関係成立届 | 適用事業になった日の翌日から10日以内 | |
| 労働保険概算保険料申告書 | 保険関係成立日から50日以内 | 概算保険料の申告・納付 |

### 5-2. ハローワーク

| 届出書類 | 期限 | 備考 |
|---------|------|------|
| 雇用保険適用事業所設置届 | 設置の日の翌日から10日以内 | 労働保険関係成立届を先に提出 |
| 雇用保険被保険者資格取得届 | 雇用日の翌月10日まで | 従業員ごとに提出 |

## 6. 届出方法の比較

| 方法 | メリット | デメリット |
|------|---------|----------|
| **窓口持参** | 不備があればその場で修正可能 | 平日日中のみ、移動時間がかかる |
| **郵送** | 時間の制約がない | 不備があると往復に時間がかかる |
| **電子申請（e-Tax等）** | 24時間対応、移動不要 | 電子証明書の取得・設定が必要 |
| **税理士・社労士に委任** | 手続きの正確性が高い、時間節約 | 報酬が発生する |

### 推奨
- 税務関係 → 税理士に依頼 or e-Tax
- 社会保険関係 → 年金事務所窓口 or 電子申請
- 都道府県・市区町村 → 郵送 or 窓口

---

## 7. 届出後の継続的義務

| 項目 | 頻度 | 期限 | 備考 |
|------|------|------|------|
| 源泉所得税の納付 | 毎月 or 年2回 | 翌月10日 or 7/10・1/20 | 特例承認の有無による |
| 法人税・消費税の確定申告 | 年1回 | 事業年度終了日から2ヶ月以内 | |
| 法人住民税の確定申告 | 年1回 | 事業年度終了日から2ヶ月以内 | |
| 法人事業税の確定申告 | 年1回 | 事業年度終了日から2ヶ月以内 | |
| 中間申告・予定納税 | 年1回 | 事業年度開始日から6ヶ月経過後2ヶ月以内 | 前期税額が20万円超の場合 |
| 年末調整 | 年1回 | 12月 | 給与所得者がいる場合 |
| 法定調書の提出 | 年1回 | 翌年1月31日 | |
| 給与支払報告書の提出 | 年1回 | 翌年1月31日 | 市区町村へ |
| 社会保険の算定基礎届 | 年1回 | 7月10日 | 標準報酬月額の見直し |
| 労働保険の年度更新 | 年1回 | 6月1日〜7月10日 | 従業員がいる場合 |
