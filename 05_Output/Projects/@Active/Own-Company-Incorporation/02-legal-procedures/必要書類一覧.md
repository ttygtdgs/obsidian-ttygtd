# 必要書類一覧

法人設立に必要な書類を、合同会社（LLC）と株式会社（Corp）別に整理する。

> ※要専門家確認: 書類の様式・必要部数は管轄の法務局・公証役場により異なる場合があります。事前に確認してください。

---

## 1. 設立前に準備する書類

### 共通（LLC・Corp）

| 書類 | 入手先 | 備考 |
|------|--------|------|
| 発起人（出資者）の印鑑証明書 | 市区町村役場 / コンビニ（マイナンバーカード） | 発行から3ヶ月以内、各1通 |
| 発起人の本人確認書類 | — | 運転免許証、パスポート等 |
| 個人の実印 | — | 印鑑登録済みのもの |

---

## 2. 定款認証に必要な書類

### 株式会社のみ（合同会社は定款認証不要）

| 書類 | 部数 | 備考 |
|------|------|------|
| 定款 | 3通 | 公証人保管用・会社保管用・法務局提出用 |
| 発起人全員の印鑑証明書 | 各1通 | 発行から3ヶ月以内 |
| 発起人全員の実印 | — | 定款に押印 |
| 委任状（代理人の場合） | 1通 | 司法書士等に委任する場合 |
| 実質的支配者となるべき者の申告書 | 1通 | 公証人法施行規則により必要 |
| 身分証明書 | — | 公証役場で本人確認 |
| CD-R / USB（電子定款の場合） | 1枚 | 電子署名済みの定款データ |

---

## 3. 登記申請に必要な書類

### 合同会社（LLC）

| 書類 | 部数 | 備考 |
|------|------|------|
| 合同会社設立登記申請書 | 1通 | 法務局の様式に従う |
| 定款 | 1通 | 社員全員の署名・押印 |
| 代表社員の印鑑証明書 | 1通 | 発行から3ヶ月以内 |
| 代表社員就任承諾書 | 1通 | 定款で定めた場合は不要 |
| 払込みがあったことを証する書面 | 1通 | 通帳コピー等を綴じたもの |
| 印鑑届出書 | 1通 | 法人実印を届出 |
| 登録免許税納付用台紙 | 1枚 | 収入印紙を貼付（6万円〜） |
| 本店所在地決定書 | 1通 | 定款で番地まで記載していない場合 |

### 株式会社（Corp）

| 書類 | 部数 | 備考 |
|------|------|------|
| 株式会社設立登記申請書 | 1通 | 法務局の様式に従う |
| 認証済み定款の謄本 | 1通 | 公証役場で取得 |
| 発起人の同意書（発起人決定書） | 1通 | 本店所在地・発行株式数等の決定 |
| 設立時取締役の就任承諾書 | 各1通 | 取締役ごと |
| 設立時代表取締役の就任承諾書 | 1通 | 取締役が複数の場合 |
| 設立時取締役の印鑑証明書 | 各1通 | 取締役会非設置会社の場合 |
| 払込みがあったことを証する書面 | 1通 | 通帳コピー等を綴じたもの |
| 資本金の額の計上に関する証明書 | 1通 | 現物出資がある場合 |
| 印鑑届出書 | 1通 | 法人実印を届出 |
| 登録免許税納付用台紙 | 1枚 | 収入印紙を貼付（15万円〜） |
| 本店所在地決議書 | 1通 | 定款で番地まで記載していない場合 |
| 設立時取締役の調査報告書 | 1通 | 現物出資がある場合 |

---

## 4. 設立後届出に必要な書類

### 税務署

| 届出書類 | 添付書類 | 提出期限 |
|---------|---------|---------|
| 法人設立届出書 | 定款の写し、登記事項証明書、株主名簿等 | 設立から2ヶ月以内 |
| 青色申告の承認申請書 | なし | 設立から3ヶ月以内 or 第1期末の前日 |
| 給与支払事務所等の開設届出書 | なし | 開設から1ヶ月以内 |
| 源泉所得税の納期の特例の承認申請書 | なし | 随時 |

### 都道府県税事務所

| 届出書類 | 添付書類 | 提出期限 |
|---------|---------|---------|
| 法人設立届出書 | 定款の写し、登記事項証明書 | 15日〜1ヶ月以内（自治体による） |

### 市区町村

| 届出書類 | 添付書類 | 提出期限 |
|---------|---------|---------|
| 法人設立届出書 | 定款の写し、登記事項証明書 | 1〜2ヶ月以内（自治体による） |

### 年金事務所

| 届出書類 | 添付書類 | 提出期限 |
|---------|---------|---------|
| 健康保険・厚生年金保険 新規適用届 | 登記事項証明書、法人番号指定通知書 | 事実発生から5日以内 |
| 被保険者資格取得届 | なし | 事実発生から5日以内 |
| 被扶養者（異動）届 | 扶養確認書類 | 事実発生から5日以内 |

### 法人口座開設

| 必要書類 | 備考 |
|---------|------|
| 登記事項証明書（履歴事項全部証明書） | 発行から6ヶ月以内 |
| 定款の写し | |
| 代表者の本人確認書類 | 運転免許証等 |
| 法人の印鑑証明書 | |
| 法人番号指定通知書 or 法人番号確認書類 | |
| 法人の実印・銀行印 | |
| 事業内容がわかる資料 | Webサイト、パンフレット等（銀行による） |

---

## 5. 書類取得コスト一覧

| 書類 | 費用 | 備考 |
|------|------|------|
| 印鑑証明書（個人） | 300〜450円/通 | コンビニだと200円の場合も |
| 登記事項証明書（法人） | 600円/通 | オンライン請求で500円 |
| 印鑑証明書（法人） | 450円/通 | オンライン請求で410円 |
| 定款認証手数料 | 30,000〜50,000円 | 株式会社のみ |
| 登録免許税（合同会社） | 60,000円〜 | 資本金の0.7% |
| 登録免許税（株式会社） | 150,000円〜 | 資本金の0.7% |

---

## 6. 推奨取得部数

設立後のさまざまな手続きで必要になるため、以下の部数を目安に取得しておくことを推奨。

| 書類 | 推奨部数 | 用途 |
|------|---------|------|
| 登記事項証明書 | 5通 | 銀行口座開設、税務署、年金事務所、都道府県、予備 |
| 法人印鑑証明書 | 3通 | 銀行口座開設、契約書、予備 |
| 定款の写し | 5部 | 各届出先に添付 |
