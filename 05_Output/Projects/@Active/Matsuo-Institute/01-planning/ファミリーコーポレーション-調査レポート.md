# ファミリーコーポレーション 総合調査レポート

> **プロジェクト目標**: ウェブサイトLP会員登録数の増加 → 不動産購入件数増 → 売上向上
> **クライアント**: 株式会社ファミリーコーポレーション
> **調査日**: 2026-02-21

---

## 1. 会社概要

| 項目 | 内容 |
|------|------|
| 社名 | 株式会社ファミリーコーポレーション |
| 設立 | 2011年4月 |
| 資本金 | 1億円 |
| 売上高 | 350億円（2025年8月、過去最高） |
| 従業員数 | 200名超 |
| 代表 | 冨吉 範明（センチュリー21 全国1位・世界7位の実績） |
| 本社 | 東京都中央区銀座6-10-1 GINZA SIX 11F |
| 拠点 | 東京・大阪・ハワイ・ロサンゼルス |

### 事業構成
- **収益不動産事業**（メイン）: 一棟アパート・マンションの売買・投資コンサル、年間200棟超販売
- **不動産BANK**: 1口1万円からの小口投資（利回り約6%）
- **土地活用事業**: 自由設計の建築ソリューション
- **ウェルスマネジメント**: 富裕層向け相続対策・資産防衛
- **海外不動産**: ハワイ（常時1,000件超）、LA
- **不動産再生・開発**: 旧耐震物件等の再生

### 強み
- **中古アパート販売実績 全国No.1（3年連続）**
- 製販一体のワンストップ体制（仕入→設計→建築→販売→管理）
- 会員数40,000名超
- 2024年度「ベストベンチャー100」選出
- 一級建築士事務所＋特定建設業許可を保有

## 2. As-Is分析（現状）

### 2-1. サイト構成の現状

```
familycorporation.co.jp（コーポレート）
├── apart.familycorporation.co.jp（アパート投資専門）★会員登録はここだけ
├── hawaii.familycorporation.co.jp（ハワイ不動産）
├── toushinohanashi.familycorporation.co.jp（投資のハナシ＝SEO記事）
├── architect.familycorporation.co.jp（建築ソリューション）
├── family-la.net（LA不動産）
├── fudousan-bank.com（不動産BANK）
└── family-property.co.jp（賃貸管理）
```

**8つのドメイン/サブドメインに分散** — ユーザー体験とデータが断片化している。

### 2-2. 会員登録導線の現状

| サイト | 会員登録導線 | 問題 |
|--------|------------|------|
| コーポレートサイト | **なし**（お問い合わせのみ） | ブランド検索流入を取りこぼし |
| アパート投資サイト | **あり**（ヘッダー＋ページ内5箇所） | 唯一の登録導線 |
| 投資のハナシ | **なし**（不動産BANKへの送客のみ） | SEO流入を全く活用できていない |
| ハワイ/LA不動産 | **なし** | クロスセル不在 |

### 2-3. 登録フォームの現状

- 必須フィールド: 7項目（氏名、メール、電話、都道府県、希望エリア、パスワード×2）
- 3ステップ構成（入力→確認→完了）
- ソーシャルログイン非対応
- **電話番号必須**（心理的ハードルが高い）

### 2-4. 現在のオファー体系

| オファー | ハードル | 金銭的インセンティブ |
|---------|---------|-------------------|
| 無料会員登録 | 中（電話番号必須） | **なし** |
| 無料投資相談 | 高（面談） | **なし** |
| セミナー参加 | 中（対面・日時限定） | **なし** |
| お問い合わせ | 高（目的不明確） | **なし** |

### 2-5. コンテンツ資産

| 種別 | 規模 | 状況 |
|------|------|------|
| コラム記事 | 179記事 | SEO集客に活用中だが会員転換弱い |
| 投資のハナシ | 多数 | 会員登録導線ゼロ |
| セミナー | 月2-3回 | **オフラインのみ**（東京・大阪） |
| 動画コンテンツ | なし | YouTube等の活用なし |
| SNS | 弱い | LINE登録導線も弱い |

### 2-6. ユーザージャーニーの断絶ポイント

```
【断絶1】Google検索 → 投資のハナシ → ??? → 離脱
         ※会員登録への導線が一切ない

【断絶2】ブランド検索 → コーポレートサイト → お問い合わせのみ → 離脱
         ※会員登録CTAが存在しない

【断絶3】セミナー参加 → ???（会員登録と別フロー） → 未会員のまま離脱
         ※セミナー→会員化のファネルが不明確

【断絶4】ハワイ/LA不動産 → 国内サイト会員登録なし → クロスセル不在
```

## 4. To-Be設計（目指す姿）

### 4-1. 全体コンセプト

**「分散する8サイトの流入を、統一された会員登録ファネルに集約し、段階的にナーチャリングして購入へ繋げる」**

```
【As-Is】                              【To-Be】
8サイトがバラバラ                        全サイトから統一ファネルへ
   ↓                                      ↓
会員登録は1サイトのみ    →    全タッチポイントに会員登録導線
   ↓                                      ↓
金銭的インセンティブなし  →    段階的インセンティブ設計
   ↓                                      ↓
オフラインセミナーのみ    →    オンライン+オフラインのハイブリッド
   ↓                                      ↓
登録後の育成不明確        →    段階的ファネル（登録→育成→面談→購入）
```

## 5. KPIツリー

```
【最終目標】売上向上
    │
    ├── 不動産購入件数 増加
    │       │
    │       ├── 面談実施数 増加
    │       │       │
    │       │       ├── ★ 会員登録数 増加（本PJの主要KPI）
    │       │       │       │
    │       │       │       ├── サイト訪問数（UU）
    │       │       │       │     ├── SEOオーガニック流入
    │       │       │       │     ├── リスティング広告流入
    │       │       │       │     ├── SNS流入
    │       │       │       │     └── リファラル流入
    │       │       │       │
    │       │       │       ├── 会員登録CVR
    │       │       │       │     ├── CTA視認率・クリック率
    │       │       │       │     ├── フォーム開始率
    │       │       │       │     ├── フォーム完了率（EFO）
    │       │       │       │     └── 導線数（設置サイト数）
    │       │       │       │
    │       │       │       └── ライト接点数（LINE/メルマガ）
    │       │       │             └── ライト → 本会員 転換率
    │       │       │
    │       │       ├── セミナー参加数
    │       │       │     ├── オンラインセミナー参加数
    │       │       │     └── オフラインセミナー参加数
    │       │       │
    │       │       └── 会員→面談予約転換率
    │       │
    │       └── 面談→成約転換率
    │
    └── 顧客単価
```

### 主要KPI定義

| KPI | 定義 | 現状推定値 | 目標値 |
|-----|------|----------|--------|
| **会員登録数** | 月間新規会員登録数 | 要計測 | 現状比 +50% |
| **サイト訪問数（UU）** | 全サイト合計の月間ユニークユーザー | 要計測 | 現状比 +30% |
| **会員登録CVR** | 訪問者→会員登録の転換率 | 要計測（推定1-2%） | 3-5% |
| **フォーム完了率** | フォーム開始→完了の割合 | 要計測 | 70%以上 |
| **ライト接点数** | LINE友だち+メルマガ登録数/月 | ほぼゼロ | 月500件 |
| **セミナー参加数** | 月間セミナー参加者数 | 約80-100名 | 月200名 |
| **面談予約率** | 会員→面談予約の転換率 | 要計測 | 10% |
| **成約率** | 面談→購入の転換率 | 要計測 | 維持or改善 |
