# ファミリーコーポレーション 売上向上ファネル分析

> **目的**: 「LP会員登録数を増やす」という施策ありきではなく、不動産投資会社の売上向上に向けてどのファネル段階に手を打つべきかを根本から整理する
> **背景**: 投資シミュレーションやチャットボットの実装が本当に売上向上に寄与するのか、そもそもの論点整理が必要

---

## 1. 売上の構造分解

### 1-1. 基本の売上公式

```
売上 = 成約件数 × 平均取引単価
```

ファミリーコーポレーションの場合（一棟アパート販売が主力）：

```
売上350億円 = 年間200棟超 × 平均1.5〜2億円/棟
```

### 1-2. 成約件数の分解

成約件数をファネルの各段階に分解すると：

```
成約件数 = 商談数 × 成約率
商談数   = リード数 × 商談化率
リード数 = サイト訪問数 × リード転換率（CVR）
```

つまり：

```
売上 = サイト訪問数 × CVR × 商談化率 × 成約率 × 平均取引単価
```

## 3. 各ステージの深掘りと論点

### ① 認知（Awareness）

**問い**: そもそもファミリーコーポレーションは十分に知られているか？

| 要素 | 現状 | 課題 |
|------|------|------|
| ブランド認知 | 「中古アパート販売No.1」だが一般認知度は低い | 競合RENOSY/PAと比べ広告投下量が少ない |
| 広告出稿 | 規模不明 | RENOSYは年間40億円、TV CM・TikTok活用 |
| SEO | 「投資のハナシ」で一定の流入あり | 記事→会員化の導線が断絶 |
| SNS | 弱い | YouTube/LINE/TikTokの活用が遅れている |
| PR | ベストベンチャー100等 | メディア露出が限定的 |

**レバレッジ度**: 中〜高（ただし広告費が大きくかかる）
**論点**: 認知拡大に投資する前に、下流のファネルが漏れバケツでないか確認が必要

---

### ② 流入（Traffic）

**問い**: サイトに来ている人は十分か？来ている人の質は適切か？

| 要素 | 現状 | 課題 |
|------|------|------|
| オーガニック流入 | 8ドメインに分散 | ドメインパワーの分散 |
| 広告流入 | 規模不明 | 費用対効果の検証が必要 |
| 流入の質 | 不明 | 「不動産投資に興味がある」層 vs 情報収集だけの層の比率不明 |
| コンテンツ流入 | 投資のハナシで集客 | 集めた人を次に繋げていない |

**レバレッジ度**: 中（量より質が重要）
**論点**: 8サイト分散は流入を分散させているだけでなく、SEO的にもドメインパワーを弱めている可能性

---

### ③ リード獲得（Lead Generation） ← 現在のPJ焦点

**問い**: 来訪者を「連絡可能な見込み客」に転換できているか？

| 要素 | 現状 | 課題 |
|------|------|------|
| 会員登録CVR | 推定1-2% | 業界平均7.4%、中央値2.6% |
| 登録導線 | apartサイトのみ | 8サイト中7サイトに導線なし |
| フォーム設計 | 7項目必須、電話番号必須 | 心理ハードル高い |
| インセンティブ | なし | 競合は2-5万円が標準 |
| ライト接点 | ほぼゼロ | LINE/メルマガの受け皿がない |

**レバレッジ度**: 高（ただし質問あり）
**重要な問い**: **CVRを上げて量を増やすことと、質の高いリードを獲得することのどちらが重要か？**

> 不動産投資は高額商材（1件1.5〜2億円）。100人の薄いリードより、10人の購入意向の高いリードのほうが成約に繋がる可能性がある。

---

### ④ 商談化（Meeting Conversion）

**問い**: リード（会員）を商談（面談）に繋げられているか？

| 要素 | 現状 | 課題 |
|------|------|------|
| 会員→面談転換率 | 要計測 | **ここの数値が最重要** |
| セミナー | 月2-3回、オフラインのみ | 地方在住者にリーチできない |
| ナーチャリング | 不明 | 会員登録後のフォロー体制が不明 |
| ISR/SDR体制 | 不明 | 電話・メールでのフォロー体制は？ |

**レバレッジ度**: 非常に高い（可能性大）
**論点**: **もし現在の会員4万人に対する商談化率が低いなら、新規会員を増やすより既存会員の商談化に注力すべき**

---

### ⑤ 提案（Proposal）

**問い**: 面談から具体的な物件提案に至っているか？

| 要素 | 現状 | 課題 |
|------|------|------|
| 面談→提案率 | 要計測 | 面談の質・営業スキルに依存 |
| 物件ラインナップ | 年間200棟超の供給力 | 供給力は強みとして十分 |
| 提案のカスタマイズ | 不明 | 顧客のニーズに合った提案ができているか |
| 物件情報の見せ方 | 会員限定物件ページ | AIレコメンドなどは未導入 |

**レバレッジ度**: 中
**論点**: ここは営業力・物件力の問題であり、デジタル施策の直接的なスコープ外

---

### ⑥ 成約（Closing）

**問い**: 提案から成約に至る率は適切か？

| 要素 | 現状 | 課題 |
|------|------|------|
| 提案→成約率 | 要計測 | 営業スキル・物件品質に依存 |
| 成約までのリードタイム | 不明 | 不動産投資は通常3-6ヶ月 |
| 競合との比較検討 | 不明 | 他社と比較されて負けるケースは？ |
| 融資支援 | あり | 強みのひとつ |

**レバレッジ度**: 中（デジタル施策では直接改善しにくい）

---

### ⑦ リピート/紹介（LTV）

**問い**: 一度購入した顧客が再購入・紹介をしてくれているか？

| 要素 | 現状 | 課題 |
|------|------|------|
| リピート購入率 | 要計測 | 1棟目→2棟目の購入率は？ |
| 顧客紹介 | 不明 | 紹介プログラムの有無 |
| 管理契約継続率 | 不明 | 管理委託のクロスセル |
| 顧客満足度 | 不明 | NPS等は計測しているか |

**レバレッジ度**: 高（高額商材ではLTVが極めて重要）
**論点**: **既存顧客4万人超から2棟目購入・紹介を促進するほうが、新規リード獲得より効率的な可能性**

## 5. 核心の問い：「会員登録数を増やす」は正しいKPIか？

### 5-1. 現状の仮説の問題点

```
現在の前提:
  会員登録数↑ → 面談数↑ → 成約数↑ → 売上↑

疑問点:
  ・既に会員4万人いるが、そのうち何%が面談に至っているのか？
  ・もし会員→面談率が1%なら、4万人×1%=400面談/累計
  ・会員を6万人に増やしても、6万×1%=600面談にしかならない
  ・しかし会員→面談率を5%に改善できれば、4万×5%=2,000面談
  ・つまり「既存会員の商談化」のほうが5倍の効果がある可能性
```

### 5-2. 確認すべき数値（松尾研/FC側に要確認）

| # | 確認事項 | なぜ重要か |
|---|---------|----------|
| 1 | **既存会員4万人のうちアクティブ会員は何人か** | 休眠会員の掘り起こし余地を判断 |
| 2 | **月間の新規会員登録数は何件か** | 現状のリード獲得力を把握 |
| 3 | **会員→面談の転換率は何%か** | ファネルのボトルネック特定 |
| 4 | **面談→成約の転換率は何%か** | 営業力の評価 |
| 5 | **成約顧客の流入経路内訳は何か** | 最も効果的なチャネルの特定 |
| 6 | **リピート購入率（1棟目→2棟目）は何%か** | LTV施策の効果予測 |
| 7 | **紹介経由の成約率は何%か** | 紹介プログラム投資の判断材料 |
| 8 | **セミナー参加→面談率は何%か** | セミナー強化の投資判断 |
| 9 | **各サイトの月間UUと流入経路** | 8サイト分散の影響度把握 |
| 10 | **営業担当1人あたりの面談キャパ/月** | リード増加しても捌けるか |

### 5-3. シナリオ別インパクト試算

現状推定値（年間200棟成約、売上350億円）をベースに、各施策のインパクトを試算：

| シナリオ | 施策 | 想定改善 | 追加成約数/年 | 追加売上/年 |
|---------|------|---------|-------------|-----------|
| A | 会員登録数を+50%増やす | 会員6万人→面談+X件 | ★面談化率次第★ | 不明 |
| B | 既存会員4万人の面談化率を2倍にする | 面談数2倍 | 最大+200棟 | 最大+350億 |
| C | セミナー月200名×面談率30% | 面談+60件/月 | +数十棟 | +数十億 |
| D | 既存顧客のリピート率を10%改善 | リピート購入増 | +20〜40棟 | +30〜80億 |
| E | 投資シミュレーション導入 | リード獲得+α | +数棟 | +数億 |

**シナリオBとDが圧倒的にインパクトが大きい可能性がある。**

## 6. 提言：ファネル分析に基づく戦略方針

### 6-1. 最優先で検証すべきこと

**「既存会員4万人の商談化率」の数値を取得する**

この1つの数値で、プロジェクトの方向性が根本的に変わる：

- **商談化率が高い場合（>5%）**: → 会員登録数を増やす方針は正しい。LP改善・フォーム改善に注力
- **商談化率が低い場合（<2%）**: → **会員を増やしても意味がない**。ナーチャリングと商談化施策が最優先

### 6-2. 暫定的な戦略優先順位

現時点で数値が不明な前提で、**最もインパクトが大きい可能性が高い順**：

```
1位: 既存会員の活性化・商談化（ファネル④）
     → 4万人のアセットを活かす。メール/LINE/セミナーで面談に誘導
     → 営業キャパとのバランスも確認

2位: 既存顧客のLTV最大化（ファネル⑦）
     → リピート購入・紹介プログラム
     → 高額商材では最もROIが高い施策

3位: リード獲得導線の最適化（ファネル③）
     → 8サイトの導線整備、フォーム改善
     → ただし量だけでなく質を重視

4位: セミナーのオンライン化（ファネル②→④の短縮）
     → 認知→商談のステップを一気に縮めるハイブリッド施策

5位: 新規施策（シミュレータ・チャットボット等）
     → 上記1〜4の基盤が整った後に検討
```

### 6-3. 投資シミュレーション・チャットボットの位置づけ

| 施策 | ファネル上の位置 | 期待効果 | 懸念 |
|------|----------------|---------|------|
| 投資シミュレーション | ③リード獲得の入口 | リード獲得の差別化ツール | 開発コスト大、リード品質不明、成約への距離が遠い |
| チャットボット型フォーム | ③リード獲得のCVR改善 | フォーム完了率の向上 | RENOSY的なUXにはなるが本質的課題（導線不足、インセンティブなし）は解消しない |

**結論**: どちらもファネル③の施策であり、**ファネル④（商談化）やファネル⑦（LTV）へのインパクトは限定的**。上流の漏れを直す前に中流・下流を固めるほうがROIが高い。

## 付録: 不動産投資会社の一般的なファネル転換率（業界参考値）

| ファネル段階 | 一般的な転換率 | 備考 |
|------------|--------------|------|
| サイト訪問→リード | 2-7% | 不動産業界平均CVR: 7.4%、中央値2.6% |
| リード→面談 | 5-15% | ISR/SDRの質に大きく依存 |
| 面談→提案 | 40-60% | 物件在庫と営業力に依存 |
| 提案→成約 | 10-30% | 高額商材のため検討期間が長い |
| 成約→リピート | 15-30% | 投資用不動産は追加購入率が比較的高い |
| 紹介発生率 | 10-20% | 満足度の高い顧客から |
