# 収益モデル設計

## 概要

5つの収益ストリームを組み合わせ、フロー型収益で短期キャッシュを確保しつつ、ストック型・スケーラブル型収益の比率を段階的に引き上げる設計。

**参照**: [[04_Memory/Business/Sales/pricing-strategy|プライシング戦略]]

---

## 1. 収益ストリーム一覧

### Stream 1: DXコンサルティング（NTTDXパートナー経由）

| 項目 | 内容 |
|------|------|
| **種別** | フロー型（案件単位） |
| **月額目安** | 50〜150万円 / 案件 |
| **粗利率** | 70-80%（業務委託活用時は50-60%） |
| **スケーラビリティ** | 中（人的リソースに依存） |
| **支払条件** | 月末締め翌月末払い |

**収益ドライバー**:
- 同時稼働案件数（Year1: 3-4件 → Year3: 5-6件）
- 案件単価の引き上げ（実績蓄積に伴う単価交渉力向上）
- 業務委託メンバーの活用によるレバレッジ

**留意点**:
- NTTDXパートナーへの依存度を65%以下に抑える計画
- 直販案件の獲得で依存度低減を図る

### Stream 2: 企業研修（直販）

| 項目 | 内容 |
|------|------|
| **種別** | フロー型（単発/短期） |
| **単価目安** | 1回60〜120万円 |
| **粗利率** | 65-75%（外部講師活用時は50-60%） |
| **スケーラビリティ** | 中（コンテンツ再利用で効率向上） |
| **支払条件** | 着手金50% + 完了時50% |

**研修メニュー**:

| メニュー | 単価 | 所要時間 | 備考 |
|---------|------|---------|------|
| AI/DXリテラシー基礎研修 | 60万円 | 半日 | 経営者/管理職向け |
| 生成AIハンズオンワークショップ | 80万円 | 1日 | 実務担当者向け |
| DX推進リーダー育成プログラム | 120万円 | 2日 | 複数回シリーズ可 |
| データ利活用研修 | 80万円 | 1日 | Looker Studio等の実習 |
| カスタマイズ研修 | 応相談 | 応相談 | 企業課題に合わせた設計 |

**参照**: [[05_Output/Areas/Business/Corporate-Training/00-service-overview|企業研修サービス概要]]

### Stream 3: メディア広告・アフィリエイト

| 項目 | 内容 |
|------|------|
| **種別** | スケーラブル型（コンテンツ資産） |
| **月額目安** | Year1: 10万円 → Year3: 80万円 |
| **粗利率** | 85-95%（ライター委託費のみ） |
| **スケーラビリティ** | 高（コンテンツ蓄積で逓増） |

**収益内訳想定**:
- Google AdSense / 純広告: 40%
- アフィリエイト（AIツール、SaaS紹介）: 35%
- スポンサード記事: 15%
- リード獲得（コンサル・研修への送客）: 10%（間接収益）

**KPI**:
- 月間PV: Year1: 5万 → Year2: 15万 → Year3: 30万
- 記事数: Year1: 100本 → Year3: 500本以上
- RPM（1,000PVあたり収益）: 200-400円

## 2. プライシング戦略

[[04_Memory/Business/Sales/pricing-strategy|プライシング戦略]] に基づき、以下の原則を適用する。

### 2.1 アンカー価格戦略
- **フロントエンド（低単価）**: 無料メディア記事 → メルマガ登録 → 低価格教材（5,000円）
- **ミドルエンド**: コミュニティ（月額2,980-9,800円）、研修（60-120万円）
- **バックエンド（高単価）**: DXコンサル月額契約（50-150万円）

### 2.2 価値ベースプライシング
- コスト積み上げではなく、顧客が得る価値（業務効率化、売上向上、補助金獲得）から逆算
- 自治体案件: 補助金活用を前提とした価格設計
- 企業研修: 参加者1人あたりのROIを可視化して提案

### 2.3 LTV最大化の導線設計
```
無料コンテンツ（メディア記事、SNS）
  ↓
リードマグネット（無料テンプレート、ホワイトペーパー）
  ↓
低単価商品（教材 5,000-30,000円）
  ↓
コミュニティ（月額2,980-9,800円）
  ↓
企業研修（60-120万円）
  ↓
DXコンサル月額契約（50-150万円/月）
```

## 5. 収益ミックスの推移

```
Year1: ██████████████░░░░░░ コンサル65% | 研修21% | スケーラブル14%
Year2: ██████████░░░░░░░░░░ コンサル53% | 研修23% | スケーラブル24%
Year3: ████████░░░░░░░░░░░░ コンサル40% | 研修25% | スケーラブル35%
```

**戦略的方向性**: 労働集約型（コンサル・研修）からスケーラブル型（メディア・教材・SaaS・コミュニティ）への段階的シフトにより、売上成長と利益率向上を同時に実現する。
