# サービスメニュー整理

## 概要

サービスを3つのTierに分類し、ストック型・フロー型・スケーラブル型のバランスで安定成長を実現する。

---

## Tier 1: ストック型（月額・継続収益）

安定的なキャッシュフローの基盤。顧客との長期関係を構築する。

### 1-1. 月額顧問コンサルティング

| 項目 | 内容 |
|------|------|
| **概要** | DX/AI活用に関する月額顧問契約 |
| **対象** | 中小企業経営者、自治体DX推進室 |
| **月額** | 50〜150万円 |
| **提供内容** | 月次定例会議、Slack/Teams相談、施策レビュー |
| **最低契約期間** | 3ヶ月 |
| **既存Areas対応** | [[05_Output/Areas/Business/Contract-Companies/|Contract-Companies]] |

**サービス詳細**:
- 月2回の定例ミーティング（各2時間）
- チャットベースの日常相談（営業日対応）
- 月次レポート作成
- 社内勉強会の企画・ファシリテーション（月1回）

### 1-2. オンラインコミュニティ

| 項目 | 内容 |
|------|------|
| **概要** | AI/DX実践者のオンラインコミュニティ運営 |
| **対象** | エンジニア、コンサルタント、DX推進担当者 |
| **月額** | 2,980〜9,800円 |
| **提供内容** | コミュニティフォーラム、月次イベント、限定コンテンツ |
| **既存Areas対応** | [[05_Output/Areas/Business/RIDE-ON-AI/00-community-strategy|RIDE ON AI]] |

**プラン構成**:

| プラン | 月額 | 内容 |
|--------|------|------|
| ベーシック | 2,980円 | コミュニティ参加、月次イベント視聴 |
| スタンダード | 5,980円 | + 質問し放題、アーカイブ全アクセス |
| プレミアム | 9,800円 | + 月1回個別相談、限定コンテンツ |

## Tier 2: フロー型（単発・プロジェクト型収益）

即座のキャッシュ創出と実績構築。コンサル・研修が中心。

### 2-1. 企業研修・ワークショップ

| 項目 | 内容 |
|------|------|
| **概要** | AI/DXリテラシー向上のための企業研修 |
| **対象** | 企業の全階層（経営者〜実務担当者） |
| **単価** | 60〜120万円 / 回 |
| **既存Areas対応** | [[05_Output/Areas/Business/Corporate-Training/00-service-overview|Corporate-Training]] |

**研修ラインナップ**:

| メニュー | 対象 | 時間 | 単価 |
|---------|------|------|------|
| 経営者向けAI/DX概論 | 経営者・役員 | 半日 | 60万円 |
| 生成AIハンズオン | 実務担当者 | 1日 | 80万円 |
| DX推進リーダー育成 | 管理職 | 2日 | 120万円 |
| データ利活用実践 | 分析担当者 | 1日 | 80万円 |
| プロンプトエンジニアリング | エンジニア | 半日 | 60万円 |
| カスタマイズ研修 | 応相談 | 応相談 | 応相談 |

### 2-2. PoC支援・プロトタイプ開発

| 項目 | 内容 |
|------|------|
| **概要** | AI/DXの概念実証（PoC）支援、プロトタイプ開発 |
| **対象** | DX導入検討中の企業・自治体 |
| **単価** | 100〜300万円 / プロジェクト |
| **期間** | 1〜3ヶ月 |

**PoC支援例**:
- 生成AIによる社内FAQ自動応答システム構築
- OCR + AI による手書き帳票デジタル化
- データ分析ダッシュボード構築（Looker Studio等）
- RPA + AI による業務自動化プロトタイプ

### 2-3. NTTDXパートナー案件

| 項目 | 内容 |
|------|------|
| **概要** | NTTDXパートナー制度経由の自治体・企業DX支援 |
| **対象** | 地方自治体、NTTグループ顧客企業 |
| **単価** | 月額50〜150万円 |
| **既存プロジェクト** | 11件以上の実績 |

**現行案件カテゴリ**:
- 自治体DX推進支援（長野県、北海道等）
- 企業DXコンサルティング（共立製薬等）
- データ利活用研修（札幌市等）

| **単価** | 5,000〜29,800円 |
| **既存Areas対応** | [[05_Output/Areas/Business/Survibe-AI-Baib-Coding-School/00-business-model|Survibe AI]] |

**教材ラインナップ想定**:

| プロンプトエンジニアリング実践 | PDF + 演習 | 14,800円 | エンジニア |

## Tier間の導線設計

```
Tier 3（スケーラブル型）: メディア記事・SNS投稿
  ↓ 認知・リード獲得
Tier 3: 無料テンプレート DL / 低単価教材購入
  ↓ 信頼構築
Tier 1: コミュニティ参加（月額2,980円〜）
  ↓ 関係深化
Tier 2: 研修・ワークショップ受注（60万円〜）
  ↓ 成果実感
Tier 1: 月額顧問契約（50万円〜/月）
  ↓ 長期パートナーシップ
Tier 2: PoC支援・大型プロジェクト（100万円〜）
```

**ポイント**: 低単価のスケーラブル商品で広く認知を取り、高単価の顧問契約・コンサル案件へとアップセルする構造。
