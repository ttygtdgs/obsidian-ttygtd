# ブランド戦略・ネーミング

## 概要

法人化に際し、ブランドアイデンティティを確立する。社名は事業ドメイン・ポジショニング・覚えやすさの3軸で検討する。

---

## 1. 社名検討フレームワーク

### 1.1 評価軸

| 評価軸 | 重み | 説明 |
|--------|------|------|
| **ドメイン適合性** | 30% | DX/AIコンサルティング事業であることが伝わるか |
| **ポジショニング** | 25% | 「地方 x 伴走型 x 実践的」のポジションが表現できるか |
| **覚えやすさ** | 20% | 発音しやすく、記憶に残るか |
| **ドメイン（Web）空き** | 15% | .com / .co.jp が取得可能か |
| **拡張性** | 10% | 将来の事業拡大（メディア、SaaS等）にも対応できるか |

### 1.2 社名候補の方向性

#### パターンA: 事業内容直結型
コンサルティング・DXを直接表現するパターン。何をやっている会社か一目瞭然。

**検討ポイント**:
- メリット: 事業内容が即座に伝わる、検索性が高い
- デメリット: 事業拡大時に名前が制約になる可能性

#### パターンB: ビジョン・コンセプト型
ミッションやビジョンから連想されるコンセプトワードを使うパターン。

**検討ポイント**:
- メリット: 事業拡大に対応しやすい、ストーリーを持たせられる
- デメリット: 初見で事業内容が分かりにくい

#### パターンC: 造語・ユニーク型
オリジナルの造語を社名にするパターン。

**検討ポイント**:
- メリット: ドメイン取得しやすい、ブランドとして独自性が高い
- デメリット: 認知に時間がかかる、意味の説明が必要

### 1.3 社名候補評価テンプレート

社名候補を以下のフォーマットで評価する。

| 候補名 | ドメイン適合 | ポジショニング | 覚えやすさ | Web空き | 拡張性 | 総合 |
|--------|:----------:|:----------:|:--------:|:-----:|:----:|:----:|
| （候補1） | /5 | /5 | /5 | /5 | /5 | /5 |
| （候補2） | /5 | /5 | /5 | /5 | /5 | /5 |
| （候補3） | /5 | /5 | /5 | /5 | /5 | /5 |

### 1.4 ネーミングチェックリスト

- [ ] 商標登録済みでないか確認（J-PlatPat）
- [ ] .com / .co.jp ドメインの空き確認
- [ ] SNSアカウント（X, LinkedIn, YouTube）の空き確認
- [ ] 類似社名がないか法人番号公表サイトで確認
- [ ] 外国語で不適切な意味にならないか確認
- [ ] 口頭で伝えやすいか（電話テスト）
- [ ] ロゴにした際のビジュアル映え

## 2. ブランドアイデンティティ設計

### 2.1 ミッションステートメント

> **AIとテクノロジーの力で、働き方と生き方を再定義し、幸せな人々を増やす**

このミッションは以下の3要素で構成される:
1. **AIとテクノロジーの力で**: 手段の明確化（DX/AI活用が事業の核）
2. **働き方と生き方を再定義**: 変革の対象（単なる業務効率化ではなく、人生そのものの変革）
3. **幸せな人々を増やす**: 社会的意義（テクノロジーは手段、幸せが目的）

### 2.2 ビジョン

> **2030年までに、地方自治体・中小企業500社以上のDX支援実績を持つ、国内有数のDX伴走パートナーとなる**

### 2.3 バリュー（行動指針）

| バリュー | 説明 | 具体的行動 |
|---------|------|----------|
| **Hands-on** | 現場に入り込み、一緒に手を動かす | 机上の空論を避け、プロトタイプを作って見せる |
| **Transparency** | 成果とプロセスを可視化する | 週次レポート、KPIダッシュボードの共有 |
| **Scalability** | 個別支援の知見をコンテンツ化し広く届ける | 研修教材化、メディア発信、テンプレート公開 |
| **Community** | 共に学び、共に成長する場を作る | コミュニティ運営、ナレッジシェア文化 |

### 2.4 ブランドパーソナリティ

| 要素 | 自社のトーン | 避けるべきトーン |
|------|-----------|---------------|
| 専門性 | 実践に基づく確かな知見 | アカデミックすぎる、理論偏重 |
| 親しみやすさ | カジュアルで分かりやすい | なれなれしい、軽すぎる |
| 信頼感 | 実績に裏付けられた安心感 | 権威的、上から目線 |
| 先進性 | 最新技術を実務に活かす | 技術オタク的、バズワード乱用 |

## 3. トーン&マナー

### 3.1 コミュニケーション原則

1. **平易な言葉で専門的な内容を伝える** — カタカナ用語の乱用を避け、具体例を多用
2. **データと事例で語る** — 抽象論よりも数字と実例
3. **「一緒にやりましょう」のスタンス** — 上から教えるのではなく、伴走する姿勢
4. **地方・中小企業への敬意** — 「遅れている」ではなく「これから伸びしろがある」

### 3.2 チャネル別トーン

| チャネル | トーン | 具体例 |
|---------|--------|--------|
| Webメディア記事 | 教育的・実践的 | 「3ステップで始める生成AI活用」 |
| SNS（X/Twitter） | カジュアル・共感型 | 「今日の研修で受講者の反応が良かった施策を共有します」 |
| 提案書・企画書 | フォーマル・信頼感 | 「当社の支援実績に基づくDX推進ロードマップをご提案します」 |
| コミュニティ | フレンドリー・対話型 | 「みなさんの現場ではこの課題、どう解決してますか？」 |
| 研修・セミナー | 分かりやすく・実践的 | 「では実際にChatGPTを使って議事録を作ってみましょう」 |

### 3.3 ビジュアルアイデンティティの方向性

| 要素 | 方向性 |
|------|--------|
| カラー | 信頼感のあるブルー系 + アクセントカラー（成長を表すグリーン or 先進性のオレンジ） |
| フォント | モダンでクリーンなゴシック体（Noto Sans JP 等） |
| ロゴ | シンプルで汎用性が高いデザイン。小さくても判別可能 |
| 写真/イラスト | 実際の研修風景、地方の企業・自治体の現場感 |

## 4. 競合との差別化ポジショニング

```
            専門特化
              ↑
              |
    技術系     |    当社の
    スタート   |    ポジション
    アップ     |      ★
              |
 安い ←-------+--------→ 高い
              |
    フリー     |    大手
    ランス     |    コンサル
              |
              ↓
            総合型
```

**自社のポジション**: 大手コンサルの品質 x フリーランスの柔軟性 x 地方特化のネットワーク

### 差別化の3本柱
1. **NTTDXパートナーの信頼性** — 大手ネットワークに裏打ちされた信頼と案件パイプライン
2. **現場伴走型の実践力** — 11件以上のDX案件で培った「手を動かせるコンサル」
3. **コンテンツ発信力** — メディア・教材・コミュニティによる知見のスケール

## 5. 社名決定: Tomoru（トモル）

### 決定事項（2026-02-15）

> **合同会社Tomoru**（トモル）

| 項目 | 内容 |
|------|------|
| **正式社名** | 合同会社Tomoru |
| **読み** | トモル |
| **由来** | 「灯る」（ともる）— 日本語の美しさ × 「Tomorrow」の未来感 |
| **コンセプト** | テクノロジーという灯火で、人々の働き方と生き方に明かりを灯す |
| **決定日** | 2026-02-15 |

### ネーミングの意味

- **灯る（ともる）**: 火が灯る、明かりが灯る。温かみと希望を表現
- **Tomorrow**: 未来への方向性、テクノロジーが切り開く明日
- **共（とも）**: 共に歩む、伴走型の姿勢
- **知（とも）**: 知識を灯す、ナレッジの伝播

### ブランドカラー方針

| 要素 | カラー | 意図 |
|------|--------|------|
| Primary | **Amber/Warm Orange** (#F59E0B → #D97706) | 灯火の温かみ |
| Secondary | **Deep Navy** (#1E293B) | テクノロジーの信頼感・深み |
| Accent | **Soft Gold** (#FCD34D) | 光の輝き、成長 |
| Background | **Warm White** (#FFFBEB → #FEF3C7) | 温もりのあるベース |
