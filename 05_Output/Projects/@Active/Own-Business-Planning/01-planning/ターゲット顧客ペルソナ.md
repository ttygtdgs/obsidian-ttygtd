# ターゲット顧客ペルソナ

## 概要

4つのペルソナを定義し、各ペルソナに最適なサービスとアプローチ方法を設計する。

---

## ペルソナ 1: 地方自治体DX推進担当者

### プロフィール

| 項目 | 内容 |
|------|------|
| **名前（仮）** | 田中 健一（45歳） |
| **役職** | 市役所 企画政策課 DX推進室 主任 |
| **所在地** | 地方都市（人口10-30万人） |
| **IT経験** | ExcelとPowerPointは使えるが、プログラミング経験なし |
| **決裁権限** | 500万円未満の予算執行権あり |

### 課題・ペインポイント
1. **DX推進計画を作ったが、実行の仕方が分からない** — 総務省のガイドラインに沿って計画は策定したが、具体的な実装方法や優先順位付けに悩んでいる
2. **庁内のデジタル人材が不足** — IT部門はシステム保守が手一杯で、新しいDXプロジェクトに割くリソースがない
3. **首長や議会への説明責任** — DX投資の効果を定量的に示す必要があるが、KPI設計の知見がない
4. **ベンダーロックインへの不安** — 大手SIerに依存すると高コストになり、内製化への移行が困難になるリスク

### ニーズ
- 「何から始めればいいか」を一緒に考えてくれる伴走パートナー
- 他自治体の成功事例を知りたい
- 職員向けのDXリテラシー研修
- 補助金・交付金の活用支援

### アプローチ方法
- **チャネル**: NTTDXパートナー制度経由、総務省/デジ庁関連セミナー登壇
- **初回接点**: 無料DX診断、他自治体事例紹介セミナー
- **提案サービス**: 月額顧問コンサル（Tier 1）、職員向けDX研修（Tier 2）
- **想定単価**: 月額80-150万円

### 購買プロセス
```
情報収集（セミナー・Webメディア）
  ↓ 2-3ヶ月
比較検討（3社以上の相見積もり）
  ↓ 1-2ヶ月
稟議・予算確保（議会承認が必要な場合あり）
  ↓ 1-3ヶ月
契約・開始
```
**リードタイム**: 4-8ヶ月（自治体は予算サイクルに注意）

## ペルソナ 2: 中小企業経営者

### プロフィール

| 項目 | 内容 |
|------|------|
| **名前（仮）** | 鈴木 美咲（52歳） |
| **役職** | 製造業 代表取締役社長（従業員50名） |
| **所在地** | 地方都市 |
| **IT経験** | スマホは使うが、社内IT環境は古い |
| **決裁権限** | 全権（オーナー経営者） |

### 課題・ペインポイント
1. **人手不足で業務が回らない** — 採用難で現場は常に人不足。残業が常態化している
2. **生成AIの話は聞くが、自社にどう活かせるか分からない** — ニュースで見るAIの話題と自社の現場が結びつかない
3. **IT投資に失敗したくない** — 過去に導入したシステムが使われなくなった苦い経験あり
4. **後継者・若手のデジタルスキル育成** — 次世代にスムーズに引き継ぐためにデジタル化を進めたい

### ニーズ
- 少額から始められるAI/DX導入支援
- 現場の業務を理解した上での提案（机上の空論はNG）
- 導入効果の可視化（コスト削減額、時間削減量）
- 補助金・助成金の活用サポート

### アプローチ方法
- **チャネル**: 商工会議所セミナー、地方銀行紹介、Webメディア
- **初回接点**: 無料AI活用診断、短時間の体験ワークショップ
- **提案サービス**: PoC支援（Tier 2）→ 月額顧問（Tier 1）
- **想定単価**: PoC: 100-200万円、月額: 50-80万円

### 購買プロセス
```
課題認識（知人の紹介、セミナー参加）
  ↓ 1ヶ月
初回相談（無料）
  ↓ 2週間-1ヶ月
PoC提案・見積もり
  ↓ 2週間
契約・開始
```
**リードタイム**: 2-3ヶ月（経営者判断で比較的早い）

## ペルソナ 3: 企業IT部門・DX推進部

### プロフィール

| 項目 | 内容 |
|------|------|
| **名前（仮）** | 佐藤 大輔（38歳） |
| **役職** | 中堅企業（従業員300名）情報システム部 課長 |
| **所在地** | 地方中核都市 |
| **IT経験** | SE出身、基本的な開発・運用スキルあり |
| **決裁権限** | 100万円未満、上位は部長決裁 |

### 課題・ペインポイント
1. **DX推進のミッションを負っているが、現場の理解が得られない** — 経営層からDX推進を任されたが、事業部門の協力が得にくい
2. **生成AIの社内ガイドライン策定** — セキュリティリスクを抑えつつ、全社的なAI活用を推進する方法が分からない
3. **最新技術のキャッチアップが追いつかない** — 日常の運用業務に追われ、新技術の調査・検証に時間を割けない
4. **社内研修の企画・実施ノウハウ不足** — 非IT部門への研修をどう設計すべきか悩んでいる

### ニーズ
- 生成AI社内ガイドライン策定の支援
- 全社向けAI/DXリテラシー研修の企画・実施
- DX推進ロードマップの策定支援
- 最新技術トレンドの情報提供

### アプローチ方法
- **チャネル**: IT系メディア、技術コミュニティ、LinkedIn
- **初回接点**: 技術ブログ記事、メディア上のホワイトペーパー
- **提案サービス**: 企業研修（Tier 2）、コミュニティ（Tier 1）
- **想定単価**: 研修: 60-120万円/回、コミュニティ: 月額5,980-9,800円

### 購買プロセス
```
情報収集（技術ブログ、メディア記事）
  ↓ 1-2ヶ月
社内起案（上長への提案書作成）
  ↓ 2-4週間
比較検討（2-3社見積もり）
  ↓ 2-4週間
稟議・承認
  ↓ 2-4週間
契約・開始
```
**リードタイム**: 3-5ヶ月

## ペルソナ 4: 個人エンジニア・フリーランスコンサルタント

### プロフィール

| 項目 | 内容 |
|------|------|
| **名前（仮）** | 山田 翔太（32歳） |
| **役職** | フリーランス Webエンジニア |
| **所在地** | 都市部（リモートワーク中心） |
| **IT経験** | 5年以上のWeb開発経験、AI/MLに興味あり |
| **可処分所得** | 月額3-5万円は自己投資に使える |

### 課題・ペインポイント
1. **AIスキルを身に付けてキャリアアップしたい** — Web開発だけでは差別化が難しくなってきた
2. **実務で使えるAI活用スキルを学びたい** — 理論よりも実践的なスキルを求めている
3. **同じ志を持つ仲間と繋がりたい** — フリーランスの孤独感、情報交換の場が欲しい
4. **コンサルタントへのキャリアチェンジ** — 開発だけでなく、上流工程やコンサルティングにも関わりたい

### ニーズ
- 実践的なAI/DX活用スキルの習得
- ポートフォリオに使える実績づくり
- コミュニティでの人脈形成
- コンサルタントとしてのスキルセット構築

### アプローチ方法
- **チャネル**: SNS（X/Twitter）、Qiita/Zenn、YouTube、コミュニティ
- **初回接点**: 無料コンテンツ（ブログ記事、YouTube動画）
- **提案サービス**: 教材購入（Tier 3）→ コミュニティ（Tier 1）
- **想定単価**: 教材: 5,000-29,800円、コミュニティ: 月額2,980-5,980円

### 購買プロセス
```
無料コンテンツの消費（SNS、ブログ）
  ↓ 1-3ヶ月
低単価教材の購入（5,000円〜）
  ↓ 1ヶ月
コミュニティ参加（月額2,980円〜）
  ↓ 継続
上位プランへのアップグレード
```
**リードタイム**: 1-3ヶ月（個人判断のため比較的早い）
