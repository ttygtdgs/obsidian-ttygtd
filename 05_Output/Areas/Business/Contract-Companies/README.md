# 契約会社管理

## 構造

```
Contract-Companies/
├─ [契約会社名]/
│  ├─ 00-契約会社概要.md      # 契約会社の基本情報
│  ├─ 01-全議事録一覧.md      # 契約会社全体の議事録一覧
│  └─ Clients/                # クライアントごとの管理
│     └─ [クライアント名]/
│        ├─ 00-クライアント概要.md
│        └─ 01-議事録一覧.md
```

## 管理ルール

### 契約会社フォルダ
- **命名**: 英語で命名（例: `NTTDX`, `Hokkaido-Team`）
- **必須ファイル**:
  - `00-契約会社概要.md` - 契約会社の基本情報
  - `01-全議事録一覧.md` - 全クライアントの議事録へのリンク集

### クライアントフォルダ
- **命名**: 日本語で命名（例: `中企庁`, `NICE`）
- **必須ファイル**:
  - `00-クライアント概要.md` - クライアントの基本情報
  - `01-議事録一覧.md` - そのクライアントの議事録へのリンク集

### 議事録の保存場所
- **プロジェクト単位**: `05_Output/Projects/@Active/[契約会社名]-[クライアント名]-[プロジェクト名]/04-delivery/`
- **ファイル命名**: `[日付]-[議題].md`（例: `2025-11-17-社外ミーティングメモ.md`）

### 議事録のリンク管理
1. **プロジェクト内**: `04-delivery/` フォルダに実際の議事録ファイルを保存
2. **クライアントの議事録一覧**: `Clients/[クライアント名]/01-議事録一覧.md` にリンクを追加
3. **契約会社の全議事録一覧**: `[契約会社名]/01-全議事録一覧.md` にリンクを追加

## ワークフロー

### 新しい議事録を作成する場合
1. プロジェクトの `04-delivery/` に議事録を作成
2. クライアントの `01-議事録一覧.md` にリンクを追加
3. 契約会社の `01-全議事録一覧.md` にリンクを追加

### 新しいクライアントを追加する場合
1. `Contract-Companies/[契約会社名]/Clients/[クライアント名]/` フォルダを作成
2. `00-クライアント概要.md` を作成
3. `01-議事録一覧.md` を作成
4. 契約会社の `00-契約会社概要.md` の「クライアント一覧」に追加
